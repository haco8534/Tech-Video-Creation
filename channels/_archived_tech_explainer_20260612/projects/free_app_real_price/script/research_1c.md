# research_1c — 歴史・設計思想・日常との接続

背景と専門家コンセンサス、そして視聴者の日常感覚との接続点。

| # | ファクト | 信頼度 | 出典 | 備考 |
|---|---|---|---|---|
| 1c-1 | ラジオ・地上波テレビはすでに「広告主がお金を払い、視聴者は無料」のモデル。つまり「無料メディアの本当の顧客は広告主」は20世紀中盤から確立している構造 | 🟢 | メディア論一般 | 1973年の "Television Delivers People" | アプリで突然起きた話ではない |
| 1c-2 | 無料アプリの台頭はスマホ普及 (iPhone 2007, Android 2008) と App Store/Play Store のオークション型広告プラットフォーム化 (2012頃) に伴う | 🟢 | 業界史 | GoogleのAdMob買収(2009)、iAd(2010) |
| 1c-3 | Zuboff の用語選択は意図的: surveillance (監視) と capitalism (資本主義) の接合。単なる「プライバシー問題」に回収されないよう、経済システムとしての構造を指すために設計されている | 🟢 | Zuboff 2014, 2019 | 感情的批判ではなく分析枠 |
| 1c-4 | 広告技術の中核は AdWords (Google, 2000) 以降の「入札による広告表示」。ユーザーの属性ではなく、その瞬間の「広告を見せる権利」に値段がつく | 🟢 | Wikipedia RTB | 「人を売る」ではなく「注意の1回分を売る」 |
| 1c-5 | SDK (Software Development Kit) はアプリに組み込まれる第三者製の部品。広告SDK・分析SDK・地図SDKなどが普通に10〜30個埋め込まれ、それぞれがデータを自社サーバに送る | 🟢 | arXiv 2024 SDK調査 | 開発者本人が全部把握していないこともある |
| 1c-6 | ある大規模調査では、Android アプリ SDK の約 37% がデータ過剰収集、iOS アプリの 98% がプライバシーマニフェストに第三者部品の記載漏れあり | 🟢 | arXiv 2409.10411, BetaNews 2025 | 制度と実態のズレ |
| 1c-7 | 日常の接続例: 天気・カーナビ・クーポン・家族位置共有アプリは位置情報アクセスを取得し、RTBパートナーに共有されやすい（EFF分析） | 🟢 | EFF 2025 | 実感しやすい身近例 |
| 1c-8 | 位置情報ブローカー Near Intelligence は RTB 経由で 10 億台以上の端末情報を集め、米国防総省・FBI・ICE・CBPに販売していたことが報道・行政資料で確認 | 🟢 | EFF, FTC | 広告技術が監視技術と地続きである実例 |
| 1c-9 | 広告枠のオークションは1件あたり数十ミリ秒で完結。入札に参加した全事業者に「誰が・どこにいて・何に興味がある」のビッドリクエストが流れる。落札者1社以外もデータを見ている | 🟢 | EFF, TrustArc | 「売る」より「配布」に近い構造 |
| 1c-10 | 専門家コンセンサス: 個人データの価値は「単体属性」ではなく「組み合わせ＋時系列＋予測可能性」で決まる。1人分を安値で売っても、数百万人の継続的な挙動観測は高値 | 🟢 | Zuboff, 学術 | なぜ1人分が安いのに全体で巨大市場なのか |
| 1c-11 | アプリが人をつなぎ止める設計は、Hook Model (Nir Eyal) — Trigger → Action → Variable Reward → Investment の循環。変動報酬が中核 | 🟢 | Eyal "Hooked" 2014 | スロットマシン理論が通知・引き下げ更新に応用 |
| 1c-12 | Fogg の B=MAP (2009) は持続的行動変容の学術モデル。善い使い方（健康・学習アプリ）と悪い使い方（依存設計）の両方に使える中立モデル | 🟢 | Fogg Behavior Model | 技術中立 |
| 1c-13 | 「ダークパターン (deceptive design)」は EU のデジタルサービス法 (DSA, 2024) や米 FTC でも規制対象に。「解約しづらい」「同意しなきゃ進めない」などが典型例 | 🟢 | deceptive.design, FTC | 設計レベルの問題として制度化中 |
| 1c-14 | 「無料アプリ」問題は、古典経済学の「外部性 externality」の一種としても読める: ユーザー個人は小さな対価（注意・データ）しか払わないが、集約された影響（注意散漫・偽情報伝播・依存）は社会に及ぶ | 🟡 | 社会経済学的視点 | 話を閉じる枠として有用 |
| 1c-15 | 日本固有の特徴: スマホ利用時間が世界最短 (1時間55分) でも、LINE・Yahoo!・PayPay のような「無料だが取引手数料モデル」の生活インフラアプリ依存度は高い。つまり「無料＝広告型」が日本で弱い側面もある | 🟢 | Statista, LINE・Z Holdings 資料 | 地域性 |
