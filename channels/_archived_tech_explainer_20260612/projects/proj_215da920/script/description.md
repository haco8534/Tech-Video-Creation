プリンターだけいつも機嫌が悪い理由

スマホもテレビも普通に動くのに、プリンターだけ「機嫌悪い」って擬人化される。この現象、実は性格じゃなくて構造の話でした。1台の家電の中で、PDL（ページ記述言語）の40年戦争、インクジェットのμmノズルと揮発性溶媒、レーザーの200℃定着ローラー、Windows NT時代から続くPrint Spooler、100年前のカミソリ商法の応用。技術の難しさ、メーカーの商売、月1しか使わない生活――3つが交差したときに「機嫌悪さ」が生まれる仕組みを、ずんだもんとめたんの対話で解きほぐします。
