## 市場寡占構造の要約（参考）

- **Kape Technologies（Teddy Sagi所有）**: ExpressVPN / CyberGhost / Private Internet Access / ZenMate / vpnMentor等レビューサイト
- **Nord Security（Lithuania、Tesonet系）**: NordVPN / Surfshark（2022年合併）/ NordPass / NordLayer
- **Proton AG（スイス、CERN出身）**: ProtonVPN / ProtonMail（独立系・無料ティア有り）

主要な「独立性ある」大手はProton AGに近く、コンシューマVPN広告市場の大部分は実質Kape vs Nord Securityの2大寡占構造。
