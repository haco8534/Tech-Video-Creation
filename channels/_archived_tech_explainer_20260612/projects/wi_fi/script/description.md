Wi-Fiが壁の向こうで弱くなる本当の話｜家の中の電波の景色

家の中でWi-Fiが強い場所と弱い場所があるのはなぜか。その答えは「壁で止まっているから」ではありません。壁は電波を止めているのではなく、通りながら弱くしているだけ。さらに、電波は家の中で壁や床に跳ね返って混ざり合い、あちこちに強弱のまだらができています。数歩動くだけで繋がり方が変わる不思議、トイレで急に弱くなる理由、2.4と5の性格の違い、そして「水分に吸われる」という伝説の真偽。最後に、ルータをどこに置けばいいのかの実用的なヒントまで、初心者向けにやさしくお話しします。
