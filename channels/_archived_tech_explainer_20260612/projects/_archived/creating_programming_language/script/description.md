# 新しいプログラミング言語を作るってどうやるの？

世界には9,000以上のプログラミング言語が存在しますが、言語はどうやって作られるのでしょうか？

字句解析、構文解析、AST、コンパイラとインタプリタの違いから、型システムやメモリ管理の設計判断まで。Python、Ruby、Go、Rustの誕生秘話とともに、プログラミング言語づくりの全体像をわかりやすく解説します。
