# Pythonの天下はいつまで続くのか

Pythonは今、プログラミング言語の中で圧倒的な人気を誇っています。
TIOBE Indexでシェア25%超、トップ大学の75%が入門コースに採用、PyPIには75万以上のパッケージ。
でも、この「天下」はなぜ実現したのか？ そしていつまで続くのか？

実はPython自身はCの100倍遅い――にもかかわらず王座にいる理由、GILという30年来の弱点、Rust・Mojo・Julia・Goという挑戦者たち、そしてCOBOLが65年経っても銀行のATMを動かし続けている事実から見える「言語の天下」の本当の意味を解説します。
