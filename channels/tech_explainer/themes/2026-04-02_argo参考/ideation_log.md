## シード分解表（6軸）

### 軸1：技術レイヤー
- ハードウェア層: CPU（AMD/Intel）、GPU（NVIDIA）、ゲーム機ハード（GB/N64）、カートリッジ vs CD
- OS層: Windows NT カーネル、メモリ保護、プロセス分離
- プラットフォーム層: ブラウザ（IE/Chrome）、Android/iOS、脱獄/root化
- アプリケーション層: UI/UXデザイン、文字エンコーディング
- セキュリティ層: 電子証明書、PKI、暗号化

### 軸2：前提・思い込み
- 「NVIDIAのGPUが最強なのは性能のおかげ」→ 実はCUDAというソフトウェアエコシステムのロックイン
- 「IEは性能が悪かったから消えた」→ 独占禁止法訴訟やWeb標準への対応遅れも大きい
- 「日本のスマホは技術力がなかった」→ 実はガラケーの機能が時代を先取りしていた
- 「Macのデザインはジョブズの天才性」→ Xerox PARCのAltoからの着想
- 「文字化けは昔のPCが古かったから」→ 文字コード規格の乱立が本質
- 「カセットは古い技術」→ ロード時間ゼロ・海賊版対策という合理的理由

### 軸3：関連する技術・概念
- PKI/電子署名 → ブロックチェーン、ゼロ知識証明、パスキー
- レトロゲーム → エミュレータ、FPGA再現、ゲーム保存問題
- GPU → 並列計算、CUDA、機械学習フレームワーク
- OS安定性 → カーネル設計、マイクロカーネル vs モノリシック
- ブラウザ → レンダリングエンジン、Web標準、Electron
- 文字コード → Unicode、絵文字の標準化、多言語処理

### 軸4：歴史・発展
- CPU: 8086 → 286 → 386 → Pentium → Core → Apple Silicon / ARM台頭
- ブラウザ: Mosaic → Netscape → IE → Firefox → Chrome → ？次は？
- ゲーム: カートリッジ → CD → DVD → BD → DL専用 → クラウド
- OS: MS-DOS → Win3.1 → 95/98(不安定) → XP(NT統合) → 現代
- スマホ: ガラケー → iPhone登場 → Android追随 → 現在の二強
- セキュリティ: パスワード → 二要素認証 → 生体認証 → パスキー

### 軸5：日常接点
- 毎日使うスマホ、PC、ブラウザ
- ゲーム体験（Switch、Steam、レトロゲーム）
- マイナンバーカード（行政手続き）
- 文字化け（メール、Webサイト）
- OSのフリーズ・ブルースクリーン体験

### 軸6：🔥 派閥・陣営
- **NVIDIA vs AMD vs Intel**: AI用GPUの選択、ゲーミングGPU
- **Apple vs Android**: エコシステム、囲い込み vs 自由
- **Windows vs Mac vs Linux**: 開発者のOS選択
- **ARM vs x86 vs RISC-V**: CPUアーキテクチャの未来
- **Chrome vs Firefox vs Safari**: ブラウザ選択、プライバシー
- **脱獄賛成派 vs 反対派**: ユーザーの自由 vs セキュリティ
- **レトロゲーム実機派 vs エミュレータ派**: 体験の本質とは
- **任天堂ハード vs PlayStation**: メディア選択の思想の違い

## テクノロジー × バズ構造スキャン（Web検索結果要約）

### 1. NVIDIA CUDA独占問題
- **論争の温度**: 🔥🔥🔥
- CUDAのロックインが独占か正当な優位かで激論中
- NVIDIA訓練市場シェア90%+予測（2025-2030）
- AMD ROCmが代替として台頭するも、ソフト互換性が壁
- Hacker News/Redditで定期的にスレッドが炎上

### 2. Chrome独占と反トラスト
- **論争の温度**: 🔥🔥🔥
- 2025年9月：裁判所がChromeの売却は不要と判断、ただし独占的検索配信契約を禁止
- 2026年上訴中。AIの台頭が判決に影響
- 「Chromeは次のIEか」論が再燃

### 3. Apple vs Android エコシステム
- **論争の温度**: 🔥🔥🔥
- 2026年AndroidのVerified Developerプログラムでサイドローディング制限→Android陣営内で論争
- iPhoneユーザーの27%がAndroidへの乗り換えを検討（2025年調査）
- EU DMA法でApple App Store独占に規制

### 4. ARM vs x86 vs RISC-V
- **論争の温度**: 🔥🔥
- RISC-Vが2026年時点でプロセッサ市場の約25%を獲得
- Apple Siliconの成功でARM優位論が強化
- x86終焉論 vs x86は死なない論で派閥形成

### 5. レトロゲーム・エミュレータの合法性
- **論争の温度**: 🔥🔥
- 任天堂 vs Yuzu訴訟（2024年和解、240万ドル）
- ゲーム保存コミュニティ vs 権利者の対立が先鋭化
- 2026年もFPGA機やエミュハンドヘルドが爆発的に売れている

### 6. Windows vs Mac vs Linux（エンジニア向け）
- **論争の温度**: 🔥🔥
- WSL2でWindows陣営が「Linuxも使える」と主張、Linux陣営は「それならLinuxでいい」と反論
- Apple Siliconで性能面のMac優位が強まる
- 2026年もSNS・掲示板で定期的に論争
