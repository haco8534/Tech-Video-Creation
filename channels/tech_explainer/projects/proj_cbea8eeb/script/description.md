MP4、AVI、MKV、WebM…動画の形式って多すぎませんか？
「とりあえずMP4を選んでるけど、本当にそれでいいの？」と思ったことがある人へ。

実は動画ファイルには「コンテナ」と「コーデック」という2つの層があり、
それぞれが独立に増え続けているからこそ、組み合わせが爆発的に多くなっています。

画質・ファイルサイズ・互換性・ライセンス料という4つのトレードオフを
各企業・団体が異なるバランスで解決しようとした結果が、今の「形式だらけ」の世界です。
