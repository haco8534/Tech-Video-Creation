# YouTube概要欄

## タイトル
無料アプリの「本当の値段」を計算してみた【ずんだもん解説】

## 概要
「タダより高いものはない」を、データと研究で科学的に検証しました。
アプリストアの97%が無料なのに、収益の98%は無料アプリから生まれている。
私たちが「データ」「注意力」「行動変容」という3つの見えない通貨で支払っている仕組みを、
0.1秒で完了するリアルタイム広告オークション（RTB）から監視資本主義まで、丁寧に解説します。
