# Design Guide — 無料アプリの「本当の値段」

## カラーパレット
テーマ：プライバシー・データ・金銭的価値

```css
:root {
    --bg: #f8f9fa;
    --text: #1a1d23;
    --text-light: #1a1d23;
    --primary: #6C3FC5;       /* ディープパープル — データ・デジタル */
    --primary-light: #ede9fe;
    --teal: #0d9488;          /* ティール — 価値・テクノロジー */
    --teal-light: #ccfbf1;
    --coral: #e11d48;         /* ローズ — 警告・コスト */
    --coral-light: #ffe4e6;
    --amber: #d97706;         /* アンバー — 注意・金銭 */
    --amber-light: #fef3c7;
    --emerald: #10B981;       /* エメラルド — ポジティブ */
    --card-bg: #ffffff;
    --border: #d1d5db;
}
```
