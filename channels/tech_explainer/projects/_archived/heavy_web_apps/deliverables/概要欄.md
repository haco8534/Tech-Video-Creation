# 最近のWebアプリ重すぎ！なぜこうなった？

Chromeのタブを10個開くだけでメモリが数GB消える。1996年のSpace Jamの公式サイトはわずか2KB、今の平均的なWebページは約2,600KB——15年で5倍以上に肥大化しました。

SPAの普及、フレームワークの多層化、依存パッケージの爆発。Webが重くなった原因を数字とともに解き明かし、HTMX・Svelte・Astroなど軽量化を目指す新しいアプローチも紹介します。
