コンパイラ vs インタプリタ
コンパイラ = 翻訳者
  • プログラム全体を事前に変換
  • 実行速度が高速
  • 変換に時間がかかる
インタプリタ = 同時通訳者
  • 1行ずつその場で実行
  • すぐに実行できる
  • 毎回通訳するため低速
