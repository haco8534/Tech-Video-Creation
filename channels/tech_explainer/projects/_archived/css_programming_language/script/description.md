## タイトル案
CSSはプログラミング言語か？ ― 30年の進化と終わらない論争

## 概要文
「CSSはプログラミング言語じゃない」――SNSで定期的に燃え上がるこの議論、あなたはどちら派ですか？

そもそも「プログラミング言語」の定義とは？ チューリング完全って何？ 1996年に生まれたCSSは30年でどこまで進化した？ calc()、CSS変数、:has()、Container Queries……現代のCSSでできる「プログラミング的なこと」を一つずつ検証しながら、この終わりなき論争の本質に迫ります。

結論は「YESかNOか」ではなく、もっと面白いところにありました。
