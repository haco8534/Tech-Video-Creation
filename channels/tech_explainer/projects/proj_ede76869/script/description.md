# YouTube概要欄

インターネットは誰が発明して、今は誰が管理しているのか？
「核戦争に備えて作られた」は本当？ 海底ケーブルの99%って？ BGPハイジャック事件って何？
実は「社長」がいないのに世界60億人がつながっている、その驚くべき仕組みを解説します。

ARPANET誕生からTCP/IP、ICANN・IETFの役割、BGPの信頼ベース設計、
そして中国のグレートファイアウォールやIANA移管まで。
インターネットの「管理」の全体像がわかる1本です。
