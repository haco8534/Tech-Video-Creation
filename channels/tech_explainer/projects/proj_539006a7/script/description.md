# YouTube概要欄

あなたが今使っているパスワード、もし8文字の数字だけだったら——最新のGPU12台でたった15分で破られます。

この動画では、パスワードの強さを決める「エントロピー」の数学、サービス側のハッシュ関数の選択で解読時間が89万倍変わる事実、そしてNISTが「定期変更は禁止」と明言した理由まで、仕組みから丁寧に解説します。

「記号を混ぜれば安全」という常識が覆る、パスワードセキュリティの本質をお届けします。
