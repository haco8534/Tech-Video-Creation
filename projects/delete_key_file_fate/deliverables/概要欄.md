# YouTube概要欄：Deleteキーを押すとファイルはどうなる？

Deleteキーを押すとファイルは「消える」──そう思っていませんか？
実は、OSはデータを消去するのではなく「参照を外す」だけ。物理データはディスク上にそのまま残っています。
この動画では、ファイルシステムの仕組みを図書館のたとえでわかりやすく解説し、HDD vs SSDの根本的な違い、デジタルフォレンジック、そして「完全に消す」方法まで深掘りします。
