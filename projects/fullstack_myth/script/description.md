# 「フルスタック」は本当に存在するのか？

求人サイトでよく見る「フルスタックエンジニア募集」。でもReactの共同作者ですら「知らないことだらけ」と告白しました。

2010年にFacebookのエンジニアが提唱した「フルスタック」は、HTML/CSS/PHP/MySQLで十分だった時代の概念。現代のスタックは100倍に膨れ上がり、全てを極めるのは物理的に不可能です。

Dan Abramovの告白、DHHの「One Person Framework」哲学、Stack Overflow調査の30.7%、GoogleやSpotifyのアプローチ、そしてT字型・π字型スキルという現実解まで。フルスタック論争の全貌を解説します。
