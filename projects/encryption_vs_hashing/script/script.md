# 【台本】暗号化とハッシュ化——似ているようで全く違う

- 作成日: 2026-03-14
- 目標尺: 約20分
- コアメッセージ: 暗号化は「元に戻せる変換」、ハッシュ化は「元に戻せない変換」。この一方通行性の違いが、パスワード保護からデジタル署名まで、セキュリティの仕組みの根幹を決定している
- 冒頭フック: もしパスワードを暗号化して保存したらどうなるか？ 多くの人が「安全だ」と思うかもしれない。でも実際には、ある有名企業がまさにそれをやった結果、1.5億人分のパスワードが丸裸にされた
