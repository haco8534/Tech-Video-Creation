# YouTube概要欄：Dockerはなぜこんなに流行ったのか

あなたのパソコンでは完璧に動くプログラムが、別のパソコンだと動かない──この「It works on my machine」問題に、世界中の開発者が何十年も苦しんできました。

実はコンテナ技術は1979年から存在していたのに、なぜDockerだけが爆発的に普及したのか？ Docker独自の3つのイノベーション（Dockerfile・Docker Hub・レイヤードイメージ）と、「技術の大成功≠企業の大成功」というDocker Inc.の意外な物語を解説します。
