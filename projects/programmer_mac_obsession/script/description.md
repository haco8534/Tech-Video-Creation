# プログラマMac好きすぎない？ — YouTube概要欄

## 📺 動画概要

なぜプログラマはMacを選ぶのか？ 「オシャレだから」では終わらない技術的・歴史的・文化的な理由を徹底解説！

macOSがUnixベースであること、1985年のジョブズ追放→NeXTSTEP開発→1996年のApple買収という歴史的ドラマ、iPhoneがもたらしたXcodeロックイン、そしてWSL2によるWindowsの逆襲まで。「じゃあ自分はどのOSを選べばいいの？」に答える20分です。
