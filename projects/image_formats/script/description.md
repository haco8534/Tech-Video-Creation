# なぜ画像形式はこんなにたくさんあるのか──「万能」が存在しない理由

JPEG、PNG、GIF、WebP、AVIF……なぜ画像形式はこんなにたくさんあるのか？
「1つに統一すればいいのに」──そう思ったことはありませんか？

でも実は、「万能な画像形式」は原理的にありえません。
この動画では、圧縮の仕組み・GIF特許騒動・次世代形式の覇権争いまで、画像形式が増え続ける本当の理由を解き明かします。
