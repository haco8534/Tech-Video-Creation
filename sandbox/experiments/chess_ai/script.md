# 【視覚的に理解する】チェスAIはどうして世界1位を倒せるのか

# ▶ 導入
めたん：チェスAIって、もう人間の世界チャンピオンより強いんでしょ？
めたん：でもさ、盤面は同じ8かける8で、駒の動きも人間と同じじゃない。
めたん：なんで機械が、世界一の人間を倒せるところまで行くの？
ずんだもん：いい疑問なのだ。
ずんだもん：チェスAIの強さは、「全手を読んでいるから」だけでは説明できないのだ。
めたん：え、全部読んでるんじゃないの？コンピュータってそういう力技のイメージだったわ。
ずんだもん：昔の強いチェスコンピュータには、かなり力技の面があったのだ。
ずんだもん：たとえば1997年にカスパロフを破った Deep Blue は、ものすごい数の局面を高速に読んだのだ。
ずんだもん：でも現代のチェスAIは、それに加えて「どの枝を読むべきか」と「局面がどれくらい良いか」をかなり賢く扱うのだ。
めたん：全部読むんじゃなくて、読む場所を選ぶのね。
ずんだもん：そうなのだ。
ずんだもん：チェスAIの中には、大きく三つの装置があると思うと見やすいのだ。
ずんだもん：一つ目は、未来の手を枝分かれで読む探索。
ずんだもん：二つ目は、その局面が有利か不利かを点数にする評価。
ずんだもん：三つ目は、対局データや自己対局から、その評価や手の選び方を強くする学習なのだ。
めたん：未来を見る目、局面を採点する目、練習で育つ目、みたいな感じ？
ずんだもん：かなり近いのだ。
ずんだもん：ということで今回は、チェスAIがどうして世界トップを倒せるのかを、AIの内部を開いて見ていくのだ。
ずんだもん：最初の問いはこれなのだ。
ずんだもん：チェスAIは、どうやって「読めないほど広い未来」から、勝ちにつながる一手を選んでいるのか。

# ▶ ボディ
## まず、チェスは未来が爆発するゲーム
ずんだもん：最初に、チェスがどれくらい大変なゲームかを見るのだ。
めたん：ルールは知ってるわよ。駒を動かして王様を詰ませるやつでしょ。
ずんだもん：ルールは短いのだ。
ずんだもん：でも未来の数がすぐ爆発するのだ。
ずんだもん：一つの局面で、合法手はだいたい数十個あるのだ。
めたん：数十個なら、人間でも考えられそうじゃない？
ずんだもん：一手だけならそうなのだ。
ずんだもん：でも自分が一手指すと、相手にも数十個の返しがある。
ずんだもん：さらにその次も数十個。
ずんだもん：これを数手先まで伸ばすと、未来の木は一気に巨大になるのだ。
めたん：枝が枝を生んで、森になるわけね。もう盤面じゃなくて林業じゃない。
ずんだもん：だから、完全に最後まで読むのは現実的ではないのだ。
ずんだもん：チェスは有限のゲームだけど、実戦中に全部読むには広すぎるのだ。
めたん：じゃあAIも、全部わかってる神様ではないのね。
ずんだもん：そうなのだ。
ずんだもん：AIも途中までしか読めない。
ずんだもん：そこで必要になるのが、途中の局面を見て「これは良さそう」「これは危なそう」と判断する力なのだ。
めたん：人間の大局観みたいなもの？
ずんだもん：近いのだ。
ずんだもん：ただしAIの大局観は、感覚ではなく数値として持つのだ。
ずんだもん：白がどれくらい良いか、黒がどれくらい良いかを、点数にするのだ。
めたん：未来の木を全部読む代わりに、途中で点数をつけるわけね。
ずんだもん：その通りなのだ。
ずんだもん：つまりチェスAIの基本は、「未来の木を少し読む」と「途中の局面を点数化する」の組み合わせなのだ。
ずんだもん：まずはこの二つを分けて見るのだ。

## 探索は、未来の枝を伸ばして刈る
ずんだもん：探索とは、今の局面から候補手を出して、その先を読むことなのだ。
めたん：候補手を全部並べて、そこから未来を伸ばすのね。
ずんだもん：そうなのだ。
ずんだもん：ただし、ただ全部を同じ深さまで読むと、すぐに限界が来るのだ。
めたん：枝が多すぎるからね。
ずんだもん：そこでチェスAIは、見込みの薄い枝を早めに捨てるのだ。
めたん：え、捨てて大丈夫なの？その枝にすごい手が隠れてたらどうするのよ。
ずんだもん：だから捨て方が重要なのだ。
ずんだもん：古典的なチェスエンジンは、minimax という考え方を使うのだ。
ずんだもん：自分は一番得する手を選び、相手は自分にとって一番嫌な手を選ぶ、と仮定して読むのだ。
めたん：相手がいつも最悪の返しをしてくる前提なのね。性格が悪いけど、勝負では正しいわ。
ずんだもん：その上で、alpha-beta という枝刈りを使うのだ。
ずんだもん：もうこの枝は選ばれないとわかったら、その先を読まないのだ。
めたん：テストで、どれだけ頑張っても合格点に届かない問題を飛ばす感じ？
ずんだもん：比喩としては近いのだ。
ずんだもん：枝刈りは「読まない勇気」なのだ。
ずんだもん：ただし雑に読まないのではなく、論理的に不要になった場所を切るのだ。
めたん：強いAIって、たくさん読むだけじゃなくて、読まない場所を決めるのもうまいのね。
ずんだもん：そこが大事なのだ。
ずんだもん：さらに現代のエンジンは、良さそうな手を先に読む順番づけ、同じ局面を再利用する表、静かな局面まで読み延ばす工夫などを組み合わせるのだ。
めたん：ただの総当たりじゃなくて、かなり整理された読み方をしているのね。
ずんだもん：そうなのだ。
ずんだもん：でも探索だけでは足りないのだ。
ずんだもん：なぜなら、どこかで読むのを止める必要があるからなのだ。
めたん：止めた場所で、局面が良いか悪いかを判断しないといけない。
ずんだもん：その通りなのだ。
ずんだもん：次は、その評価の中身を見るのだ。

## 評価関数は、盤面を点数に変える目
ずんだもん：評価関数は、局面を点数に変える装置なのだ。
めたん：駒得してたらプラス、王様が危なかったらマイナス、みたいな？
ずんだもん：最初のイメージはそれでいいのだ。
ずんだもん：昔ながらのチェスエンジンでは、駒の価値、キングの安全、ポーンの形、駒の働き、中央支配などを人間が設計していたのだ。
めたん：つまり「チェスの先生が採点表を作った」みたいなものね。
ずんだもん：そうなのだ。
ずんだもん：Deep Blue も、強力な探索と、人間の知見を入れた評価関数で強かったのだ。
ずんだもん：ただし、この方式には限界もあるのだ。
めたん：採点表にない良さを見落とすとか？
ずんだもん：まさにそれなのだ。
ずんだもん：たとえば、しばらく駒を損しても、相手のキングを縛れるなら良い場合がある。
ずんだもん：一見静かな手が、十手後の圧力を作っていることもある。
めたん：人間でも難しいやつだわ。ぱっと見で損に見える好手。
ずんだもん：そこでニューラルネットが入ってくるのだ。
ずんだもん：ニューラルネット型の評価器は、盤面を数字の入力にして、「この局面はどちらがどれくらい良いか」を学習するのだ。
めたん：採点表を人間が全部書くんじゃなくて、局面から学んで採点するのね。
ずんだもん：そうなのだ。
ずんだもん：現代の Stockfish は、NNUE という軽いニューラルネット評価を探索の中で使うのだ。
ずんだもん：NNUE は、チェスの一手で盤面の一部だけが変わることを利用して、評価を高速に更新できるのだ。
めたん：ニューラルネットって重そうなのに、探索の中で何度も使えるようにしてあるのね。
ずんだもん：そこがすごいのだ。
ずんだもん：強い探索は大量の局面を見る。
ずんだもん：だから評価器は賢いだけでなく、速くなければならないのだ。
めたん：賢いけど一局面に一分かかります、じゃ使えないものね。
ずんだもん：その通りなのだ。
ずんだもん：チェスAIの評価器は、盤面を「人間が読める説明」ではなく、「探索で使える点数」に変換する目なのだ。
ずんだもん：では次に、AlphaZero 系のAIが何を変えたのかを見るのだ。

## AlphaZeroは、評価と候補手を自己対局で育てた
ずんだもん：AlphaZero が面白いのは、人間の棋譜や定跡を中心に覚えたわけではないところなのだ。
めたん：ルールだけ知って、あとは自分で指しまくったって話よね。
ずんだもん：そうなのだ。
ずんだもん：最初はほぼ何も知らない状態から、自己対局を繰り返す。
ずんだもん：勝った局面、負けた局面、良かった手を材料にして、ニューラルネットを更新するのだ。
めたん：自分で問題を作って、自分で解いて、自分で採点するみたいで怖いわね。
ずんだもん：怖いけど、ゲームではかなり強いのだ。
ずんだもん：なぜならチェスには明確な勝敗があるからなのだ。
ずんだもん：最後に勝ったか負けたかという信号を、途中の判断へ戻せるのだ。
めたん：勝敗という答え合わせがあるから、練習が回るわけね。
ずんだもん：AlphaZero のネットワークは、大きく二つのことを出すのだ。
ずんだもん：一つは policy、つまり次にどの手が有望そうか。
ずんだもん：もう一つは value、つまりこの局面が最終的に勝ちに近いか。
めたん：候補手の地図と、局面の採点を同時に出すのね。
ずんだもん：そうなのだ。
ずんだもん：そして探索では、この policy が読む枝の優先順位を決め、value が枝先の良し悪しを見積もるのだ。
めたん：未来の森を、AIの地図を持って歩く感じね。
ずんだもん：かなり近いのだ。
ずんだもん：AlphaZero 系では、Monte Carlo Tree Search という探索を使うのだ。
ずんだもん：全部の枝を同じように読むのではなく、有望そうな枝へ探索回数を集めるのだ。
めたん：強そうな道に人数を多く送り込むわけね。
ずんだもん：そうなのだ。
ずんだもん：ここで重要なのは、ニューラルネットが一手を直接決めて終わりではないことなのだ。
ずんだもん：ネットワークの直感と、探索による検証を組み合わせるのだ。
めたん：直感だけでも、力技だけでもない。直感で候補を絞って、探索で裏を取るのね。
ずんだもん：そこが現代的な強さなのだ。
ずんだもん：だから AlphaZero は、少ない探索数でも鋭い手を見つけやすいのだ。
ずんだもん：人間の定跡を暗記したからではなく、自己対局で「勝ちに近い形」を内部に作ったからなのだ。

## 人間とAIでは、見ているものが違う
めたん：でも人間のトップ棋士だって、何十年も勉強してるわけでしょ。
めたん：AIの何がそんなに違うの？
ずんだもん：大きな違いは、疲れずに大量の候補を検証できることなのだ。
ずんだもん：人間は、経験と直感で候補手をかなり絞る。
ずんだもん：そして深く読める局面もあるけど、全部の候補を均等には読めないのだ。
めたん：人間は集中力も時間も限られるものね。あとお腹も空くし。
ずんだもん：AIはお腹が空かないのだ。
ずんだもん：さらにAIは、気まずい手を避けないのだ。
めたん：気まずい手？
ずんだもん：人間には「この形は変」「こんな駒損は怖い」「普通はこう指す」という感覚があるのだ。
ずんだもん：それは強い武器だけど、時には偏見にもなるのだ。
めたん：常識が、見えない壁になることがあるのね。
ずんだもん：そうなのだ。
ずんだもん：AIは、見た目が変でも、探索と評価で良いなら候補に残す。
ずんだもん：長期的な圧力、キングの安全、逃げ道の少なさ、相手の駒の働かなさを、点数と探索で扱うのだ。
めたん：だから「なんでそんな手？」と思ったら、何手もあとに効いてくるわけね。
ずんだもん：そうなのだ。
ずんだもん：ただしAIも万能ではないのだ。
ずんだもん：評価器が苦手な形、探索が届かない罠、持ち時間やハードウェアの制限はあるのだ。
めたん：それでも人間より強いのは、弱点が実戦で出る前に、だいたい読めてしまうから？
ずんだもん：その理解でいいのだ。
ずんだもん：人間は「意味」で盤面を見る。
ずんだもん：AIは、意味っぽい特徴も含めて、膨大な候補を数値と探索で見る。
ずんだもん：見ている世界が違うのだ。

## 何がすごいのかは、「知識」より「改善ループ」
めたん：ここまで聞くと、チェスAIの中にはすごいチェス知識が詰まってるってこと？
ずんだもん：結果としてはそう見えるのだ。
ずんだもん：でも本当にすごいのは、知識を直接書き込んだことではなく、改善ループを作ったことなのだ。
めたん：改善ループ？
ずんだもん：候補手を出す。
ずんだもん：探索する。
ずんだもん：評価する。
ずんだもん：対局結果で学習する。
ずんだもん：強くなったモデルで、さらに強い自己対局をする。
ずんだもん：この循環なのだ。
めたん：強くなった自分が、次の先生になるのね。
ずんだもん：そうなのだ。
ずんだもん：AlphaZero や Leela Chess Zero の発想は、まさにそれなのだ。
ずんだもん：一方で Stockfish も、分散テストで小さな改善を大量に検証し、NNUE の評価を鍛え続けているのだ。
めたん：ニューラルネット型も、伝統的エンジン型も、結局は試して強くする文化なのね。
ずんだもん：その通りなのだ。
ずんだもん：チェスAIの強さは、単発の天才プログラムではないのだ。
ずんだもん：探索、評価、学習、テストが回り続ける仕組みなのだ。
めたん：つまりAIが世界トップを倒す理由は、「賢い評価器を持った探索マシンが、無限に近い練習で磨かれるから」？
ずんだもん：かなり良いまとめなのだ。
ずんだもん：最後に、最初の問いへ戻るのだ。

# ▶ 結論
ずんだもん：最初の問いは、チェスAIはどうやって「読めないほど広い未来」から勝ちにつながる一手を選ぶのか、だったのだ。
ずんだもん：答えは、未来を全部読むのではなく、探索で枝を伸ばし、不要な枝を刈り、評価関数で途中の局面を点数化するからなのだ。
めたん：そして現代のAIは、その評価や候補手の選び方を、ニューラルネットや自己対局で育てているのね。
ずんだもん：そうなのだ。
ずんだもん：Deep Blue は、巨大な探索と人間知識入りの評価で世界チャンピオンを倒した。
ずんだもん：AlphaZero は、自己対局で policy と value を育て、探索と組み合わせて超人的な手を見つけた。
ずんだもん：現代の Stockfish は、alpha-beta 探索と NNUE 評価を組み合わせ、膨大なテストで磨かれているのだ。
めたん：一種類の魔法じゃなくて、世代ごとに強さの作り方が進化してるのね。
ずんだもん：その通りなのだ。
ずんだもん：チェスAIの中にあるのは、盤面を完全に理解する小さな名人ではないのだ。
ずんだもん：未来の木、局面を点数化する目、勝敗から自分を直すループなのだ。
めたん：だから人間が見落とす変な手も、AIには勝ち筋として見えることがある。
ずんだもん：なのだ。
ずんだもん：チェスAIが世界トップを倒せる理由は、人間よりチェスを愛しているからではないのだ。
ずんだもん：人間には読み切れない未来を、探索と評価と学習のループで、勝ちに近い形へ圧縮できるからなのだ。
