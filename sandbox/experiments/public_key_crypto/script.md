# 公開鍵暗号 ── なぜ鍵を渡さずに秘密を送れるのか

# ▶ 序論

めたん：ねえずんだもん。ネットでお買い物するとき、カードの番号を打ちこむでしょう。
ずんだもん：打ちこむのだ。番号を入れて、送信ボタンを押すのだ。
めたん：あの番号、お店に届くまでのあいだ、ほかの誰にも見られていないのかしら。
ずんだもん：いい質問なのだ。とちゅうの通り道は、じつはけっこう、人目のある場所なのだ。
めたん：人目のある場所。
ずんだもん：送り手と受け手のあいだには、長い通信路が伸びているのだ。番号はそこを通るのだ。
めたん：その通信路を、誰かがのぞいていることもある、ってこと？
ずんだもん：のぞける場所なのだ。通る中身は、その気になれば読み取られてしまうのだ。
めたん：えっ。じゃあ、わたしのカード番号、まる見えじゃない。
ずんだもん：ところが、ちゃんと隠して送れているのだ。今日は、その隠し方の話なのだ。
めたん：隠して送る……箱に入れて、鍵をかけて送れば安心かしら。
ずんだもん：すなおな考えなのだ。箱に入れて、錠前をかけて、送る。
めたん：それなら、のぞかれても中身は読まれないわね。
ずんだもん：でも、受け取った人は、その箱をどうやって開けるのだ？
めたん：あっ……鍵もいっしょに送らないと、開けられないわ。
ずんだもん：その鍵も、おなじ通信路を通るのだ。のぞいている人に、鍵まで見られるのだ。
めたん：鍵を見られたら、箱を開けられてしまう。隠した意味がないのね。
ずんだもん：秘密を送るのに、まず鍵という秘密を届けないといけない。にわとりと卵なのだ。
めたん：先に鍵を安全に渡せるなら、その同じ道で秘密も渡せてしまうものね。
ずんだもん：そのとおりなのだ。なのに今、初対面の相手とも秘密をやりとりできているのだ。
めたん：会ったこともないお店と、こっそり話せている。……どうやっているの？
ずんだもん：その仕掛けに、名前があるのだ。公開鍵暗号、というのだ。
めたん：公開鍵暗号。
ずんだもん：鍵を渡さずに、秘密を送る。その不思議を、これからほどいていくのだ。

# ▶ ボディ

## 錠前と鍵を、別々のものにする

ずんだもん：まず、思いこみをひとつ外すのだ。錠前と、その鍵を思いうかべるのだ。
めたん：南京錠みたいなものよね。鍵で開けて、鍵で閉めるもの。
ずんだもん：そこなのだ。錠前は、閉めるのにも鍵が要ると思われがちなのだ。
めたん：ちがうの？
ずんだもん：パチンと押しこめば、それだけで掛かる錠前があるのだ。閉めるのに、鍵は要らないのだ。
めたん：あー、あるわね。押すと、かちゃっと閉まるやつ。
ずんだもん：開けるときだけ、鍵が要る。閉める人と、開ける人は、べつの人でいいのだ。
めたん：閉めるのは誰でもできて、開けられるのは鍵を持つ人だけ。
ずんだもん：この「閉める」と「開ける」を切り分けたのが、公開鍵暗号の出発点なのだ。
めたん：切り分けると、何がうれしいのかしら。
ずんだもん：受け手が、錠前と鍵をひとそろい用意するのだ。そして、鍵は手元に置いておくのだ。
めたん：鍵は、自分だけが持っておくのね。
ずんだもん：いっぽうの錠前は、開いたまま、いくつも通信路に流してしまうのだ。
めたん：錠前を、配ってしまうの？のぞいている人にも届いてしまうわよ。
ずんだもん：届いていいのだ。錠前は、配るためのものなのだ。だから公開鍵と呼ぶのだ。
めたん：のぞき見している人も、わたしと同じ錠前を持つことになるけれど……いいの？
ずんだもん：いいのだ。錠前を持っていても、できるのは「閉める」ことだけなのだ。
めたん：あっ。錠前があっても、開けることはできないのね。
ずんだもん：送り手は、配られた錠前で箱をパチンと閉じて、送り返すのだ。
めたん：閉めるのに鍵は要らないから、錠前さえあれば送り手は誰でも閉められる。
ずんだもん：のぞいている人にも、その箱は見えるのだ。でも、開ける鍵は持っていないのだ。
めたん：開けられるのは、鍵を手元に残した受け手だけ。
ずんだもん：手元の鍵で、かちゃり。鍵そのものは、一度も通信路を通っていないのだ。
めたん：渡したのは錠前だけ。鍵は、ずっと受け手の手のなかにあったのね。
めたん：わたしのお部屋の鍵も、町じゅうに配って回れば無くさず済むのかしら。
ずんだもん：手元に残すこの鍵を、秘密鍵と呼ぶのだ。配る錠前が公開鍵、残す鍵が秘密鍵なのだ。
めたん：公開鍵で閉めて、秘密鍵で開ける。鍵を渡さずに、秘密が送れたわ。

## 閉める計算と、開ける計算

めたん：でも、ひとつ引っかかるの。錠前を配るなら、その仕組みも相手に知られるでしょう。
ずんだもん：いい引っかかりなのだ。錠前を調べれば、開け方も分かりそうなものなのだ。
めたん：錠前から鍵を逆算されたら、おしまいじゃない。
ずんだもん：そこで、錠前の正体を見るのだ。錠前の中身は、金属ではなく、ひとつの計算なのだ。
めたん：計算。
ずんだもん：箱を閉じるというのは、中の数に、ある決まった計算をほどこすことなのだ。
めたん：その計算をやり返して元の数に戻せば、それが「開ける」ということね。
ずんだもん：ところがこの計算は、ある向きには進みやすく、逆向きには進みにくいのだ。
めたん：片道だけ、やさしい道。
ずんだもん：たとえば、大きな素数をふたつ、掛け合わせるのだ。掛け算じたいは、すぐ終わるのだ。
めたん：素数は、それ以上われない数よね。ふたつ選んで掛けるくらいは、できるわ。
ずんだもん：では逆に、掛けた答えだけを渡して、元のふたつの数を言い当ててもらうのだ。
めたん：答えから、元のふたつ……ひとつずつ割れるか試していくしかないのかしら。
ずんだもん：それなのだ。数が十分に大きいと、その総当たりは、とほうもない時間がかかるのだ。
めたん：掛けるのは一瞬。もとに戻すのは、気が遠くなるほど大変。おなじ計算の行き帰りなのに。
ずんだもん：この行き帰りの差が、錠前の正体なのだ。閉めるのはやさしい行き、開けるのは険しい帰り。
めたん：では、鍵を持つ受け手も、その険しい帰り道を歩いているの？
ずんだもん：そこが鍵の正体なのだ。鍵は、その険しい帰り道に付いた、近道なのだ。
めたん：近道。
ずんだもん：受け手だけが知っている数があると、帰り道に隠し扉が開くのだ。落とし戸と呼ぶのだ。
めたん：その隠し扉を通れば、戻る計算が、すうっとやさしくなる。
ずんだもん：錠前を調べても、落とし戸の合言葉は出てこないのだ。だから逆算できないのだ。
めたん：のぞいている人には、険しい道しかない。受け手にだけ、近道がある。
ずんだもん：念のため言うと、「絶対に解けない」のではなく「現実的な時間で解けない」なのだ。
めたん：うんと速い計算機が現れたら、話は変わるかもしれない、ということね。
ずんだもん：そのとおりなのだ。だから使う数の大きさは、時代に合わせて見直されているのだ。

## 同じ鍵で、送り主を確かめる

ずんだもん：ここまでで、秘密は送れたのだ。でも、この鍵のひとそろいには、もう一芸あるのだ。
めたん：まだあるの？
ずんだもん：さっきは、配った錠前で閉めて、手元の鍵で開けた。今度は、順番を逆さにするのだ。
めたん：鍵で閉めて、錠前で開ける……そんなこと、できるのかしら。
ずんだもん：できるのだ。受け手が、手元の秘密鍵のほうで、箱を閉じてしまうのだ。
めたん：秘密鍵で閉めた箱。それは、誰が開けられるの？
ずんだもん：配られた錠前——公開鍵を持つ人なら、世界じゅうの誰でも開けられるのだ。
めたん：誰でも開けられたら、秘密にならないじゃない。
ずんだもん：今度のねらいは、隠すことではないのだ。確かめることなのだ。
めたん：確かめる。何を？
ずんだもん：その箱が、配られた錠前できれいに開いたとするのだ。そこから、何が言えるのだ？
めたん：その錠前と、ひとそろいの秘密鍵で閉じられた、ということ。
ずんだもん：そして、その秘密鍵を持っているのは、世界でただ一人なのだ。
めたん：あっ。開けられたという事実が、そのまま「あの人が閉めた」という証しになるのね。
ずんだもん：これを署名と呼ぶのだ。中身を隠していた錠前が、今度は送り主の印になったのだ。
めたん：おなじひとそろいが、秘密を守る向きと、本人を証す向きの、両方に使えるのね。
ずんだもん：閉めると開けるが別の手の仕事だからこそ、向きを変えるだけで役目も変わるのだ。
めたん：サインはまねされたら困るけれど……この署名は、まねできないのかしら。
ずんだもん：まねるには秘密鍵が要るのだ。そしてそれは、誰の手にも渡っていないのだ。
めたん：わたしのサインも、練習帳ごと隠しておけば、まねされず済むのかしら。
ずんだもん：受け取った人は、配られた錠前で開くかどうかを見るだけで、本物かどうか分かるのだ。

# ▶ 結論

ずんだもん：通信路ぜんたいを、もう一度たどってみるのだ。
ずんだもん：はじまりは、人目のある通信路。秘密を送りたいのに、鍵という秘密が先に要った。
めたん：にわとりと卵。秘密を渡すのに、秘密が要る、堂々めぐり。
ずんだもん：そこで、錠前と鍵を、べつべつの手の仕事に切り分けたのだ。
めたん：閉めるのは誰でも。開けるのは、鍵を持つ人だけ。
ずんだもん：開いた錠前——公開鍵を配り、開ける鍵——秘密鍵だけを手元に残す。
めたん：錠前の中身は、片道の計算。行きはやさしく、帰りは険しい。鍵だけが近道を持つ。
ずんだもん：そして同じひとそろいは、向きを逆さにすれば、送り主の署名にもなったのだ。
めたん：はじめの不思議に、戻ってきたわね。
ずんだもん：「鍵を渡さずに、秘密を送れるのか」という不思議なのだ。
めたん：……渡す必要は、はじめから無かったんだ。渡したのは、錠前だけ。
ずんだもん：閉めることと開けることは、はじめから別の手の仕事だったのだ。
めたん：のぞかれてもいい錠前と、たった一つの鍵。それで、見知らぬ相手とも秘密を結べる。
ずんだもん：鍵を「配るもの」と「隠すもの」に分けた——それが、公開鍵暗号のお話だったのだ。
