# zip爆弾

# ▶ 序論
めたん：ねえずんだもん。これ、たった42キロバイトの小さなファイルなの。
ずんだもん：ああ、それ、開けちゃだめなやつなのだ。ひらいた瞬間、4.5ペタバイトに化けるのだ。
めたん：ペタバイト？
ずんだもん：ペタは、ギガの百万倍なのだ。スマホ数百万台ぶんが、その小さな一個から膨らむのだ。
めたん：そんなの、入りきらないでしょう。
ずんだもん：入らないのだ。だから開けたパソコンは、ディスクを食いつぶして潰れるのだ。
めたん：たった42キロから？
ずんだもん：そう、たった42キロからなのだ。これがzip爆弾、ぶっ壊れた縮め方の産物なのだ。
めたん：zip爆弾。
ずんだもん：ふしぎなのは、爆弾そのものは、指一本ぶんも重くないってことなのだ。
ずんだもん：重い思いをするのは、開けた側だけなのだ。作るのは軽くて、開けるほうが重い。
めたん：軽いものが、開けた相手だけを潰すの……？
ずんだもん：そこが今日のキモなのだ。なぜそんなことが起きるのか、ほどいていくのだ。

# ▶ ボディ

## 圧縮はデータでなく指示書
ずんだもん：まず、ふつうのファイルがどうやって小さくなるか、見るのだ。圧縮ってやつなのだ。
めたん：ぎゅっと縮める、あれよね。
ずんだもん：縮めるっていうより、書き直すのだ。「あ」が千個ならんだ紙を、思いうかべてほしいのだ。
めたん：「あ」が千個。
ずんだもん：そのまま書けば千文字なのだ。でも「あ、を千回」と書けば、たった数文字で済むのだ。
めたん：あっ、すごく短くなった。
ずんだもん：これが圧縮の正体なのだ。中身そのものじゃなく、書き方の指示だけを残すのだ。
めたん：じゃあ、圧縮ファイルの中身って……。
ずんだもん：データじゃなく、指示書なのだ。「ここを見て、これだけ繰り返せ」という命令の束なのだ。
めたん：でも、繰り返す元のデータは、どこかに持ってるんでしょう？
ずんだもん：ほんの少しでいいのだ。「直前の並びを、また千回」と指させば、元はひと並びで足りるのだ。
ずんだもん：この「前を指して繰り返せ」の仕掛けを、LZ77と呼ぶのだ。zipの心臓、DEFLATEの土台なのだ。
めたん：エルゼットなんとか。
ずんだもん：だから繰り返しだらけの紙ほど、指示は短くなる。中身が単調なほど、爆弾は軽くなるのだ。

## 一枚の指示書では爆弾になれない
ずんだもん：ところが、指示書ひとつには、ちゃんと天井があるのだ。
めたん：天井？
ずんだもん：DEFLATEの指示書は、どうあがいても、千三十二倍までしか膨らませられないのだ。
めたん：千三十二倍。けっこう大きいじゃない。
ずんだもん：42キロを千倍しても、43メガちょっとなのだ。今どきのディスクは、それじゃ潰れないのだ。
めたん：あれ、爆弾にならないわね。
ずんだもん：そこで昔の人が考えたのが、指示書のなかに、また指示書を入れることなのだ。
めたん：指示書のなかに、指示書。
ずんだもん：開けたら、また圧縮ファイル。それを開けたら、また圧縮ファイル。これを五段、重ねるのだ。
ずんだもん：一段で十六個ずつ増やすと、五段で何倍になると思うのだ。百倍か、百万倍か。
めたん：……百万倍の、ほう？
ずんだもん：そう、ざっと百万倍なのだ。これが有名な42.zip、42キロが4.5ペタに化ける正体なのだ。
めたん：さっきの4.5ペタって、これだったの？
ずんだもん：そうなのだ。でも入れ子には、尻尾があるのだ。開けても開けても、また圧縮ファイルが出てくる。
めたん：それ、見たらバレちゃわない？
ずんだもん：バレるのだ。「また圧縮ファイルが出たら、もう開かない」。検査する側は、そう身構えたのだ。
ずんだもん：入れ子の爆弾は、こうして見破られた。爆弾は、一段に戻るしかなくなったのだ。

## 重ねずに、重ねる
ずんだもん：ここからが新しい爆弾なのだ。2019年に、フィフィールドって人が示したやり方なのだ。
めたん：入れ子を、使わずに？
ずんだもん：使わないのだ。一段のまま、巨大にするのだ。カギは、zipの目次なのだ。
めたん：目次。
ずんだもん：zipは、中身の一覧表を持っているのだ。「一個目はここ、二個目はここ」と、場所を指す目次なのだ。
めたん：本の目次と、おなじね。
ずんだもん：そこなのだ。ふつうは、目次の項目それぞれが、別々の中身を指すのだ。
ずんだもん：フィフィールドは、目次の項目を、ぜんぶ同じひとかたまりに向けたのだ。
めたん：ぜんぶ、おなじ場所を指すの……？
めたん：でも、おなじ中身を何度も数えるなんて、ずるじゃない。
ずんだもん：数える側は、それを別々の中身だと思いこんで、律儀に全部展開してしまうのだ。
ずんだもん：ひとかたまりの圧縮データを、千個の項目で指す。一個ぶんの重さが、そのまま千倍に展開されるのだ。
めたん：層は、一枚のままなの？
ずんだもん：一枚のままなのだ。だから「また圧縮ファイルが出る」尻尾が無い。検査の身構えを、すり抜けるのだ。
ずんだもん：この一段の爆弾は、46メガで4.5ペタを生むのだ。入れ子とおなじ威力を、隠れたまま出すのだ。
めたん：おなじ4.5ペタを、見つからずに。

# ▶ 結論
ずんだもん：もう一度、爆弾の正体を見るのだ。
ずんだもん：中身は、ほとんど無い。あるのは、これを繰り返せという、短い指示だけなのだ。
めたん：それだけ。
ずんだもん：書くのは一瞬なのだ。なのに開けた側は、その指示を最後まで律儀に実行して、自分の資源を焼くのだ。
めたん：えっ、ちゃんと開けたほうが？
ずんだもん：そうなのだ。爆弾は、自分では何もしない。ただ、相手の律儀さを、武器にするだけなのだ。
ずんだもん：これ、生きものの世界にもあるのだ。ウイルスは、自分だけでは増えないのだ。
めたん：ウイルス？
ずんだもん：ウイルスは、増え方を書いた指示書を、細胞に持ちこむだけなのだ。あとは細胞が、律儀に読んで増やすのだ。
めたん：指示書を実行する側が、自分で増やしちゃうのね。
ずんだもん：ばか正直に指示を実行するもの。その真面目さが、そのまま弱点になるのだ。
ずんだもん：いちばん軽いものが、いちばん重い結果を生む。zip爆弾は、たった数十キロの、そんなお話だったのだ。
めたん：……いちばん、軽いのにね。
