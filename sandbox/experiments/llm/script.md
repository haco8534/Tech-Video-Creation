# ▶ ボディ

## 言葉を地図の上に置く

ずんだもん：まず、コンピュータは数字しか扱えないのだ。「りんご」って文字のままじゃ、計算できないのだ。
めたん：じゃあ、辞書の順に番号をつければ？
ずんだもん：それが素朴な第一案なのだ。りんご1番、みかん2番、政府3番……。でもこれ、致命的にダメなのだ。
めたん：えっ、どうして。
ずんだもん：1番のりんごと、2番のみかん。番号が近いから意味も近い、とはならないのだ。たまたま隣に並べただけなのだ。
めたん：あ……番号って、近い順に並んでるわけじゃないのね。
ずんだもん：そうなのだ。しかも番号だと、足し算しても意味がないのだ。りんごの1に、みかんの2を足して、3の政府になっても、なんにも嬉しくないのだ。
めたん：たしかに、めちゃくちゃね。
ずんだもん：じゃあ、意味の近さを、どうやって数字にするか。ここで発想を変えるのだ。
めたん：どうするの？
ずんだもん：地図を思いうかべてほしいのだ。広い平面に、ぜんぶの単語を点として置くのだ。
ずんだもん：似た意味の言葉は近くに、ちがう意味の言葉は遠くに置く。りんごとみかんは隣どうし、政府はずっと向こうなのだ。
めたん：意味の地図、ね。
ずんだもん：この「単語を場所として持つ」やり方を、埋め込み、分散表現って呼ぶのだ。LLMの土台なのだ。
ずんだもん：あ、正確には単語まるごとじゃなく、トークンっていう部品で切るんだけど、ここは単語で話すのだ。
めたん：場所にすると、何がいいの？
ずんだもん：ただ近い遠いだけじゃないのだ。地図の上だと、「向き」にも意味が乗るのだ。
めたん：向き？
ずんだもん：有名な例があるのだ。「王様」の場所から、「男」のぶんだけ引いて、「女」のぶんだけ足すのだ。どこに着くと思うのだ。
めたん：……どこなの？
ずんだもん：「女王」の場所に、ぴたっと着くのだ。
めたん：えっ、計算で？
ずんだもん：計算で、なのだ。「男から女へ」っていう向きが、地図のどこでも同じだけ使えるのだ。意味の関係が、矢印になってるのだ。
めたん：場所だけじゃなく、向きにも……。
ずんだもん：そう。番号にはこんな芸当、できないのだ。場所にしたとたん、意味の足し引きができるようになる。これがエグいのだ。
めたん：番号と、ぜんぜん違うのね。
ずんだもん：でも、ここで一番の疑問なのだ。誰が、その場所を決めるのだ？
めたん：たしかに、誰が決めるの？
ずんだもん：いい質問なのだ。そこがいちばん面白いとこなのだ。人間が手で決めるのは、無理なのだ。
めたん：無理？
ずんだもん：単語は何万個、地図は何百、何千っていう次元なのだ。ぼくらの三次元の、何千倍もの方向に広がってるのだ。
めたん：何千次元！？
ずんだもん：なんでそんなに要るかっていうと、意味って、いろんな面を持ってるからなのだ。
めたん：いろんな面？
ずんだもん：「りんご」は、食べ物で、赤くて、丸くて、果物で、わりと安い。一個の言葉に、測る物差しがいくつもあるのだ。
めたん：あ、たしかに何通りもある。
ずんだもん：その物差し一本いっぽんが、地図の一次元なのだ。測る面がたくさんあるから、次元もたくさん要るのだ。
めたん：物差しの数だけ、次元が。
ずんだもん：そんな空間の、何万個ぜんぶに手で座標をつけるなんて、できないのだ。だから、言葉自身に決めさせるのだ。
めたん：言葉自身に？
ずんだもん：「りんご」って言葉は、どんな言葉と一緒に出てくると思うのだ。
めたん：えーと、食べる、甘い、皮をむく……。
ずんだもん：それなのだ。「みかん」の周りにも、似たような言葉が並ぶ。でも「政府」の周りは、ぜんぜんちがう言葉なのだ。
ずんだもん：だから決め方はこうなのだ。周りに出てくる言葉が似てる単語どうしを、近くに置く。それだけなのだ。
めたん：周りの言葉が似てたら、近くに……？
ずんだもん：言語学者のファースが、こう言ったのだ。「言葉は、つきあう仲間を見ればわかる」。1957年の言葉なのだ。
めたん：つきあう仲間で、決まるの？
ずんだもん：そうなのだ。意味そのものは、一文字も使ってない。周りにどんな言葉が来るか、その並びぐせだけで場所が決まるのだ。
めたん：並びぐせだけで……。
ずんだもん：しかも、こう置けって教えるわけじゃないのだ。やらせるのは、ただの穴埋め問題なのだ。
めたん：穴埋め？
ずんだもん：「むかしむかし、あるところに、おじいさんと」の最後を隠して、当てさせるのだ。外したら、外したぶんだけ、地図を少し動かすのだ。
めたん：間違えながら、直していくのね。
ずんだもん：それを、何十億回もくりかえすのだ。読ませる文章は、本にして何千万冊ぶん。人が一生かけても、読みきれない量なのだ。
めたん：何千万冊……。
ずんだもん：そのうちに、りんごとみかんを近くに置いたほうが、穴埋めがよく当たる。だから、自然と寄っていくのだ。
めたん：当てるために、寄っていくの？
ずんだもん：そうなのだ。誰も座標を教えてないのだ。次の単語を当てる練習だけで、意味の地図が、ひとりでに出来あがる。これがエグいのだ。
めたん：勝手に、地図ができる。
ずんだもん：これ、ぼくらが言葉を覚えるやり方と、まるで同じなのだ。子どもは「犬」の意味を、辞書で習わないのだ。
めたん：あ、たしかに。
ずんだもん：散歩、吠える、しっぽ。そういう言葉と一緒に何度も聞くうちに、犬の場所が決まっていくのだ。
めたん：わたしも、そうだったのかしら。
ずんだもん：これで単語が一個ずつ、地図の上に場所を持ったのだ。……でも、ここで困ったことが起きるのだ。
めたん：困ったこと？
ずんだもん：「冷たい水」と「冷たい人」。同じ「冷たい」が、場所一個で足りると思うのだ？
めたん：あ……ちがう意味よね。

## 文脈で意味を研ぎ澄ます

ずんだもん：さっきの地図だと、「冷たい」は場所が一個きりなのだ。水の冷たいも、人の冷たいも、同じ点なのだ。
めたん：同じ言葉なんだから、一個でいいんじゃないの？
ずんだもん：それだと区別できないのだ。「銀行」と「土手」、英語だとどっちも同じbankなのだ。
めたん：つづりが、同じ……。
ずんだもん：場所が一個きりなら、銀行の点と土手の点が、ぴったり重なってしまうのだ。どっちの意味か、わからないのだ。
めたん：困るわね。
ずんだもん：それに、こんなのもあるのだ。「猫が魚を食べた。それは新鮮だった」。この「それ」、何を指すと思うのだ。
めたん：……魚、よね。
ずんだもん：魚なのだ。でも「それ」っていう単語の場所は、一個きり。文だけ見ても、魚なのか猫なのか、決まらないのだ。
めたん：あ、ほんとだ。
ずんだもん：昔のやり方は、文を前から順に、一語ずつ読んでいくものだったのだ。
めたん：ふつうに、左から読む。
ずんだもん：でもそれだと、ずっと前に出た言葉を、読んでるうちに忘れちゃうのだ。「それ」まで来たころには、「魚」が遠いのだ。
めたん：人間でも、長い文だと忘れるものね。
ずんだもん：だから発想を変えたのだ。前から順に読むのをやめて、文の全部の語を、一度にぜんぶ見渡すのだ。
めたん：いっぺんに、ぜんぶ？
ずんだもん：いっぺんに、ぜんぶなのだ。文の中の一語が、ほかの全部の語に「きみは、ぼくに関係ある？」って聞いて回るのだ。
めたん：関係あるか、聞いて回る。
ずんだもん：「それ」は聞いて回って、「魚」が一番関係あるって気づくのだ。そして魚の意味を取りこんで、自分の場所を「魚」の側へ動かすのだ。
めたん：点が、動くのね。
ずんだもん：「冷たい水」の「冷たい」も同じなのだ。近くの「水」に関係ありって気づいて、温度の冷たさの側へ寄るのだ。
ずんだもん：「冷たい人」なら、「人」に寄って、性格の冷たさの側へ動く。この見渡して注目する仕掛けを、Attention、注意機構って呼ぶのだ。
めたん：アテンション。
ずんだもん：これがTransformer、いまのLLMの心臓なのだ。全部の語が全部の語に聞いて回るなんて、ちょっと反則みたいな力技なのだ。
めたん：ぜんぶが、ぜんぶに。
ずんだもん：誰にどれだけ注目するか、その聞き方も、さっきの穴埋めで覚えるのだ。「それ」の次は近くの名詞を見ろ、ってだんだん上手くなるのだ。
めたん：注目の仕方も、練習で。
ずんだもん：ただ、全部が全部を見るのは、タダじゃないのだ。文が二倍長くなると、見るべき組み合わせは、四倍に増えるのだ。
めたん：四倍。けっこう増えるのね。
ずんだもん：だから、一度に見渡せる長さには限りがあるのだ。それを超えた古い話は、すーっと忘れていくのだ。
めたん：人間と、似てるわね。
ずんだもん：ここで一個、当ててみてほしいのだ。「彼の話は、いつも明るい」。
めたん：うん。
ずんだもん：この「明るい」は、部屋の明るさと、性格の明るさ、どっちに寄ると思うのだ？
めたん：……性格の、ほう？
ずんだもん：正解なのだ。近くに「話」「彼」がいるから、明るいの点はそっちへ動く。光のほうには寄らないのだ。
めたん：周りが、決めてくれるのね。
ずんだもん：おもしろいのは、最初はどっちつかずで、点がぼやけてることなのだ。文脈が来て、はじめて一点に決まるのだ。
めたん：ぼやけてたのが、ピントが合う感じ。
ずんだもん：その言いかえ、ぴったりなのだ。文脈っていうピントで、ぼやけた意味が一点に合うのだ。
めたん：写真の、ピント合わせみたい。
ずんだもん：そうなのだ。しかもこれを、一回じゃなく何十回も重ねるのだ。寄って、また見渡して、また寄って。だんだん意味が研がれていくのだ。
めたん：何回も、研ぐのね。
ずんだもん：おもしろいのは、これで初めて見る文でも、ちゃんと扱えることなのだ。
めたん：初めての文？
ずんだもん：知らない文も、知ってる言葉の場所を、文脈で混ぜ合わせれば、どこかに置けるのだ。丸暗記じゃないのだ。
めたん：見たこともない質問に答える、って……。
ずんだもん：それが、これなのだ。決まった答えを覚えてるんじゃなく、その場で組み立ててる。だから、初めての問いにも答えるのだ。
めたん：暗記じゃ、なかったのね。
ずんだもん：これで、文脈で研ぎ澄まされた点ができたのだ。文の最後の点には、それまでの文がぜんぶ吸いこまれてるのだ。
めたん：全部が、その一点に。
ずんだもん：そう。じゃあ最後に、その点から「次の単語」をどう取り出すのか。最初の謎に、戻るのだ。

## 次の一語を引いて、また引く

ずんだもん：文脈を吸いこんだ、最後の点。ここから、次に来そうな単語ぜんぶに、確率をふるのだ。
めたん：確率？
ずんだもん：「私はラーメンを」の続きを考えるのだ。次に来るのは「食べた」か、「書いた」か。どっちが高いと思うのだ？
めたん：それは「食べた」でしょう。
ずんだもん：そうなのだ。食べたが60パーセント、すすったが10、作ったが5……語彙ぜんぶに、確率がつくのだ。
めたん：書いた、はほぼゼロね。
ずんだもん：ほぼゼロなのだ。何万語の中で、ありそうなところに山ができる。で、その山にしたがって、一個だけ選ぶのだ。
めたん：いちばん高いのを選ぶの？
ずんだもん：たいていは高いやつ、でもたまに、低いのも選ぶのだ。わざと、少しゆらすのだ。
めたん：わざと、外すの？
ずんだもん：毎回いちばん高いのだけ選ぶと、文章がかちかちに固くなるのだ。同じこと聞いたら、毎回そっくり同じ答えなのだ。
めたん：それは、つまらないわね。
ずんだもん：このゆらぎ具合は、つまみで変えられるのだ。低くすれば、かっちり正確。高くすれば、自由で創造的になるのだ。
めたん：へえ、調整できるのね。
ずんだもん：でも、ここに落とし穴があるのだ。LLMが平気で嘘をつく、あれの正体なのだ。
めたん：嘘？
ずんだもん：LLMは「知らない」と「それっぽい」の区別が、ないのだ。やってるのは、ずっと、次に来そうな単語を選ぶことだけなのだ。
めたん：あ……知らなくても、選んじゃう。
ずんだもん：選んじゃうのだ。事実かどうかなんて、どこにも見てない。それっぽく続く単語が、自信たっぷりに出てくるのだ。
めたん：賢く見えるのと、同じ仕掛けなの？
ずんだもん：同じ仕掛けで、なのだ。賢さと、でたらめは、同じ一つの仕組みの裏表なのだ。
めたん：分けられないの……？
ずんだもん：分けられないのだ。さあ、ここからが肝心なのだ。一語選んだら、終わりじゃないのだ。
めたん：終わりじゃない？
ずんだもん：選んだ「食べた」を、文のお尻にくっつけるのだ。「私はラーメンを食べた」。そしたら、また全部を読み直すのだ。
ずんだもん：読み直して、その次の一語を、また確率で引く。引いたらくっつけて、また読み直す。これを延々くりかえすのだ。
めたん：一語ずつ、また最初から？
ずんだもん：一語ふえるたびに、ぜんぶ読み直すのだ。この「自分の出力を、また自分に食わせる」やり方を、自己回帰って呼ぶのだ。
めたん：自分のを、自分に。
ずんだもん：しかも、一度出した単語は、引っこめられないのだ。書き直しのきかない、一発勝負の喋りなのだ。
めたん：あとから、直せないの？
ずんだもん：直せないのだ。落語家が、つっかえずに即興でしゃべり続けるのに、近いのだ。先を決めずに、口だけが先に進むのだ。
めたん：考えてから、話してるわけじゃ……。
ずんだもん：ないのだ。でも、頭の中ではぜんぶ考えてから書いてるんでしょう？って、思うのだ。
めたん：うん、思う。
ずんだもん：それが、無いのだ。文章全体の設計図なんて、どこにも持ってない。一語先しか、見てないのだ。
めたん：あっ、さっき言ってた。
ずんだもん：なのに、段落が、筋の通った話が、動くコードまで出来あがるのだ。設計図なしで、一歩ずつ積むだけで、建物が建つのだ。
めたん：一歩先しか見てないのに……。
ずんだもん：そこが、いちばん化け物じみてるとこなのだ。先のことは何も決めてない。なのに後から振り返ると、ちゃんと筋が通ってるのだ。

# ▶ 結論

ずんだもん：もう一度、LLMの中身を見るのだ。意味も、知識も、入ってないのだ。
めたん：ほんとに、空っぽ？
ずんだもん：空っぽなのだ。あるのは、どの言葉がどの言葉の隣に出るか。膨大な文章から測った、場所と距離だけなのだ。
めたん：場所と、距離だけ。
ずんだもん：「りんごとは何か」を、LLMは知らないのだ。ただ「りんご」が、どの言葉の近くに置かれるかを、知ってるだけなのだ。
めたん：じゃあ、なんで答えられるの？
ずんだもん：哲学者のサールが、昔こんな話をしたのだ。中国語の部屋、っていうのだ。
めたん：中国語の部屋？
ずんだもん：部屋に、中国語をまったく知らない人がいるのだ。でも分厚いマニュアルがあって、この記号が来たらこの記号を返せ、と書いてあるのだ。
めたん：意味は、わからないまま。
ずんだもん：わからないまま、規則どおり記号を返すのだ。すると外の人には、中の人が中国語を理解してる、としか見えないのだ。
めたん：外からは、理解してるように見えちゃう？
ずんだもん：見えちゃうのだ。サールは、これは理解じゃない、ただの記号の操作だ、と言ったのだ。LLMは、まさにこの部屋なのだ。
めたん：意味を知らずに、記号を……。
ずんだもん：でも、ここで思いだしてほしいのだ。「言葉は、つきあう仲間で決まる」。あれ、もとは人間の話だったのだ。
めたん：あ、ファースの。
ずんだもん：ぼくらも、言葉の意味を辞書から覚えたわけじゃないのだ。周りでどう使われてるかで、覚えてきたのだ。
めたん：……同じ材料を、使ってるってこと？
ずんだもん：そうなのだ。意味を理解してるはずの人間と、場所を計算してるだけのLLM。使ってる材料は、おどろくほど同じなのだ。
めたん：理解と、計算の境目が……。
ずんだもん：思ったより、ずっと薄いのだ。LLMが理解してないなら、同じやり方で覚えたぼくらは、本当に理解してるのか。
めたん：えっ、こっちが問われてる。
ずんだもん：意味がわかるってことが、本当は何なのか。次の単語を当てる機械は、こっちに問い返してくるのだ。
めたん：当ててるだけ、なのにね。
