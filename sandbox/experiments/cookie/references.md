# 参考文献 - Cookieってなんなの？？

- Using HTTP cookies - MDN Web Docs, https://developer.mozilla.org/en-US/docs/Web/HTTP/Guides/Cookies: Cookieの基本用途、`Set-Cookie` と `Cookie` ヘッダー、寿命、`Secure`、`HttpOnly`、`SameSite`、Cookie prefixes の説明に使用。
- Set-Cookie header - MDN Web Docs, https://developer.mozilla.org/en-US/docs/Web/HTTP/Reference/Headers/Set-Cookie: `Domain`、`Path`、`Expires`、`Max-Age`、Cookie prefixes、`Partitioned` 属性の具体例と制約の確認に使用。
- Cookies: HTTP State Management Mechanism, IETF HTTP Working Group, draft-ietf-httpbis-rfc6265bis-22, 2026確認, https://datatracker.ietf.org/doc/html/draft-ietf-httpbis-rfc6265bis: Cookieの仕様上の受信、属性処理、寿命制限、ドメイン処理、SameSiteなどの根拠として使用。
- Third-party cookies - Privacy Sandbox, Google, last updated 2025-12-11 UTC, https://privacysandbox.google.com/cookies: 第三者Cookie、CHIPS、`Partitioned` Cookie、Storage Access API、第三者Cookie制限の現在の整理に使用。
- Update on Plans for Privacy Sandbox Technologies - Google Privacy Sandbox, 2025-10-17, https://privacysandbox.google.com/blog/update-on-plans-for-privacy-sandbox-technologies: Chromeが第三者Cookieのユーザー選択を維持しつつ、CHIPSやFedCMを継続し、一部Privacy Sandbox技術を廃止する文脈の確認に使用。
- OWASP Session Management Cheat Sheet, https://cheatsheetseries.owasp.org/cheatsheets/Session_Management_Cheat_Sheet.html: セッションIDの扱い、Cookie属性、セッション固定や失効など、ログインCookieを認証情報として扱う説明の補助に使用。
