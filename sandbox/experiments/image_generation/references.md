# 参考文献

- Learning Transferable Visual Models From Natural Language Supervision — Alec Radford et al., OpenAI, 2021, https://arxiv.org/abs/2103.00020: 画像と文章を同じ表現空間に置くCLIPの説明の根拠
- Denoising Diffusion Probabilistic Models — Jonathan Ho, Ajay Jain, Pieter Abbeel, 2020, https://arxiv.org/abs/2006.11239: ノイズを加えて戻す拡散モデルの基本説明の根拠
- High-Resolution Image Synthesis with Latent Diffusion Models — Robin Rombach et al., 2022, https://arxiv.org/abs/2112.10752: 潜在空間で画像生成を行い、cross-attentionで条件付けする説明の根拠
- SDXL: Improving Latent Diffusion Models for High-Resolution Image Synthesis — Dustin Podell et al., Stability AI, 2023, https://arxiv.org/abs/2307.01952: 高解像度の潜在拡散モデルと複数テキストエンコーダの発展を説明する補助資料
- Scaling Rectified Flow Transformers for High-Resolution Image Synthesis — Patrick Esser et al., Stability AI, 2024, https://arxiv.org/abs/2403.03206: 拡散以後の流れとして、Rectified FlowとTransformer化を説明する根拠
- Introducing 4o Image Generation — OpenAI, 2025, https://openai.com/index/introducing-4o-image-generation/: 会話文脈、編集、文字描画、instruction followingの近年の変化を説明する根拠
- Image generation — OpenAI API Documentation, 2026確認, https://platform.openai.com/docs/guides/image-generation: GPT Image系の生成・編集API、会話内生成、制約やモデル世代を確認する資料
