# diffusion — design overall

`script.md` を入力に、動画全体を貫通する世界観と、6画面それぞれのデザインを自然言語で設計する。座標値・フレーム数・セリフ単位のタイミングは持たない（design_spec と Composition.tsx の仕事）。

図解は概念に**ラベルを貼る**のではなく、仕組みを**動いて見せる**。「拡散モデル」と書いた箱は出さない。絵がノイズへ崩れ、ノイズから絵が立ち上がる過程そのものを画面の上で起こす。本作の図は、汎用の箱・矢印の寄せ集めではなく、**ひとつの仕組みをひとつ可視化する専用装置**の集まりとして設計する。

## 背骨

貫通する背骨は **キャンバス**——絵の場。細かなタイルの格子でできた一枚の窓で、内部に「絵」「砂あらし（ノイズ）」、あるいはその中間を映す。キャンバスは **ノイズ量** という一つの数値を持つ。0なら鮮明な絵、1なら完全な砂あらし、その間は絵がノイズに溶けた途中の姿になる。

動画全体は、このノイズ量が「絵 → 砂あらし（壊す）」「砂あらし → 絵（逆からのぼる）」「でたらめな砂あらし → 注文どおりの絵（言葉で誘導）」へと変化していく一本の物語になる。個々の装置——ノイズ階段・ノイズ予測器・ずれメーター・引き算記号・方位磁針・強度つまみ——は、すべてこのキャンバスのノイズ量を上げ下げし、その下げる向きを決めるための道具として連なる。装置はバラバラの飾りではなく、ひとつのキャンバスを加工する一連の工程として互いにつながっている。

## 1. 世界観

白テーマ（背景・面が白系）と文字サイズ下限は固定ベース（→ `02_design_overall.md`「固定ベース」）。比喩・図の語彙・アクセント色を、拡散モデル固有に設計する。

| 軸 | この題材での設定 |
|---|---|
| 貫通する比喩 | **明るい現像台**。やわらかい白の作業台の上に、キャンバスが一枚据えられている。砂あらしは台の上で粒立ち、絵は暖かな色でそこに浮かび上がる。霧が晴れて像が結ぶ——写真が現像されるように、絵がノイズの中から立ち現れる。視聴者は同じ台をずっと眺めながら、一枚のキャンバスが壊され・読まれ・組み直されていく全工程を追う |
| 形の語彙 | 専用の図解装置の集まり（一覧は §3 図カタログ）。背骨は **キャンバス**＝タイル格子の窓。映る「絵」は単純なドット絵（ねこ・宇宙服のねこ・家・花・山・顔。暖色の少数色で組む）。映る「砂あらし」はタイルごとにばらばらなグレーの粒。装置（ノイズ階段・ノイズ片・ノイズ予測器・ずれメーター・引き算記号・方位磁針・強度つまみ）は、キャンバスとノイズ片を入力に「何かを計算する／向きを変える道具」として設計する。汎用の「ラベル付き箱」「意味のない矢印」「テキストだけのカード」は使わない |
| 色の語彙 | 背景・面は白テーマで固定（背景 `#f5f7fa`／パネル面 `#ffffff`／枠 `#c4cedd`／主文字 `#243044`）。砂あらし＝グレーの粒（暗〜明のランダム）。絵＝暖色少数色：ねこ毛のオレンジ／陰影の焦げ茶／鼻のピンク／クリーム白／空の水色。アクセント（題材ごと）：機械（ノイズ予測器）＝ティール `#1aa39a`（白地で締まる寒色、絵の暖色と役割が衝突しない）。プロンプト・方位磁針・誘導＝アンバー `#e0901e`（「行き先を決める」働きの色）。ずれ・誤差＝コーラル `#d9543c`。階段・補助線・退いた要素＝グレー |
| 動きの語彙 | 使う＝フェード、ノイズ量の連続変化（タイルが絵↔砂あらしの間で滑らかに溶け合う）、階段を一段ずつ下る／のぼる、ノイズ片が装置から湧き出て移動する、方位磁針の針が向きを変える、つまみが目盛りを滑る、収束。封印＝3D回転、カメラのズーム／パン、バウンド、急な点滅、**ループし続ける動き（浮遊・振動・明滅・脈動・常駐背景モーション）**。装置は会話の進行に合わせてイベントごとに一度きり動き、終わったら止まる（合間は静止していてよい） |
| 文字方針 | 画面下部に**対話字幕**を常駐（話者名は話者色、本文は濃い藍、スマホ視聴前提で大きめ・2行まで）。図解内のテキストは最小限：ノイズ階段の段番号、ノイズ予測器のラベル、ずれメーターの目盛り、プロンプト文字列「宇宙服を着たねこ」、強度つまみの「弱／強」。サイズは固定ベースの下限を守る |
| ナビ補助 | 画面が切り替わるとき、左上に小さくセクション番号と見出し（例「01 絵を、わざと壊す」）。進捗ドット等は置かず最小限。字幕帯とは別の位置 |

レイアウトの基準：中心原点。**画面下部 y≈330〜510 は対話字幕の帯**として確保し、主オブジェクトはそれより上に置く。下端に重要物を置かない。主役のキャンバスは上寄り〜中央、補助の階段・装置はその下や脇。1画面に装置を全部いっぺんに出さない——会話が進むにつれ装置が順に立ち上がり、役目を終えたら畳まれ、次の装置に場所を譲る。

## 2. セクション別デザイン

### 序論（画面1）

- **登場する図**：主役のキャンバス1枚、人の描き方（紙＋一本ずつ増える線）、砂あらし、ドット絵のねこ、虫めがね、タイトル「拡散モデル」
- **レイアウト**：キャンバスを中央やや上に大きく1枚。説明の対比物（線を足す手の動き）はキャンバスの面の上で起こす。タイトルはキャンバスの下、字幕帯の上の余白に。
- **動き**：キャンバスが現れ、まっさらな明るい紙を映す → 紙の上に線が一本ずつ足されて簡単な家の絵になる（人の描き方＝足し算）→ 線がすっと消え、キャンバス全面が砂あらしに変わる（AIの出発点）→ 砂あらしのノイズ量がすこしずつ下がり、霧が晴れるようにねこのドット絵が立ち現れる（引き算で描く）→ 虫めがねが砂あらしの一点に寄るが、そこには粒しかない（奥に絵は無い、という不思議）→ キャンバスが砂あらしと絵の中間の姿でとどまり、その下にタイトル「拡散モデル」が灯る
- **絵の眼目**：絵は紙に線を足して描くのではなく、砂あらしからノイズを引いて現れる。だが砂あらしの奥に絵は無い——これが解くべき不思議

### ボディ1：絵を、わざと壊す（画面2）

- **登場する図**：主役のキャンバス（鮮明なねこ）、振りかけられるノイズ、ノイズ階段（崩れ具合の違う小キャンバスの列）、記録ノイズ片（各段に付く、その段で足した砂）
- **レイアウト**：主役のキャンバスを上部中央に大きく。ノイズ階段はその下に、左（鮮明）から右（完全な砂あらし）へ下りていく小キャンバスの横列。記録ノイズ片は階段の各段の真上に小さく添える。
- **動き**：キャンバスが鮮明なねこを映す → ノイズがぱらっと振りかけられ、キャンバスのノイズ量がわずかに上がる。同時に、いまの姿が小キャンバスとして階段の一段目に写し取られる → もう一回、さらに一回。振りかけるたびにキャンバスがざらつき、階段が一段ずつ右下へ伸びる → 何段ぶんか進み、最後の段は完全な砂あらし。主役のキャンバスも砂あらしになる → 「壊し方のこだわり」：階段の隣り合う二段を並べて見せ、一段の差はごく小さいことを示す（小さな一歩なら後で戻せる）→ 「記録」：階段の各段の上に、その段で足したノイズの記録片が一枚ずつ灯る
- **絵の眼目**：鮮明な絵が、小さなノイズの一段を何十回も重ねて砂あらしになる。その一段一段は記録される

### ボディ2：ノイズを、言い当てる練習（画面3）

- **登場する図**：出題キャンバス（ノイズの乗った絵）、ノイズ予測器（機械）、予測ノイズ片（機械の答え）、記録ノイズ片（控えてある正解）、ずれメーター、修正の表示、反復カウンタ、出題に流れこむ写真サムネ群（ねこ・犬・山・顔）
- **レイアウト**：上辺に出題のもとになるノイズ階段。出題キャンバスを中央左。ノイズ予測器を中央。機械から出る予測ノイズ片を機械の右上に、正解の記録ノイズ片をその下に並べ、二枚の下にずれメーターを横たえる。反復カウンタは出題キャンバスの上辺寄り。
- **動き**：ノイズ階段の途中の一段（半分壊れたねこ）が出題キャンバスとして中央左へ据えられる → ノイズ予測器が出題キャンバスを読み、右へ予測ノイズ片を一枚はき出す（「足されたノイズはこれ」という答え）→ その下に、控えてあった記録ノイズ片（正解）が灯る → 二枚を見くらべ、下のずれメーターの針がそのずれ幅まで振れる → 修正：機械の中身がわずかに整い、ずれメーターの針がほんの少し戻る → 反復：出題キャンバスに犬・山・顔の写真が次々と切り替わり、予測→答え合わせ→修正が高速でくりかえされる。反復カウンタが跳ね上がり、ずれメーターの針が回を追うごとに小さく収まっていく → ノイズ予測器が安定して光り、技が定着したことを示す
- **絵の眼目**：出題→答え合わせ→修正を何百万回くりかえし、機械は「ノイズを言い当てる」たった一つの技を身につける

### ボディ3：壊れ方を、逆からたどる（画面4）

- **登場する図**：主役のキャンバス、ノイズ予測器（前画面の技を持ち越し・小型）、予測ノイズ片、引き算記号、のぼり階段、学習で見た写真の淡い記憶（背後の写真サムネ群）、出発を変えた数組（砂あらし→絵）
- **レイアウト**：主役のキャンバスを中央左。ノイズ予測器はキャンバスの右脇に小さく控える。のぼり階段はキャンバスの下に、右下（砂あらし）から左上（絵）へのぼる小キャンバスの列。複数の結果を見せる場面では、出発の砂あらしと到達の絵を数組、横に並べる。
- **動き**：砂あらしのキャンバスにノイズ予測器が向き、予測ノイズ片を出す → 引き算記号でキャンバスから予測ノイズ片が引かれ、ノイズ量が一段ぶん下がる（階段を一段のぼる）→ 予測→引く→のぼる、を一段ずつくりかえし、キャンバスがすこしずつ鮮明になる → のぼりきってノイズ量が0になり、元のねこが戻る → 出発点をすりかえる：まっさらな別の砂あらしのキャンバスが据えられる（元の絵は無い）→ 同じ予測→引く→のぼるをくりかえすと、見たことのない新しい絵がぼうっと浮かび上がる → 背後に、機械が学習で見てきた写真サムネが淡くよぎり、ノイズを引くたびキャンバスが「より絵らしいほう」へ寄ることを示す → 出発の砂あらしを変えた数組を横に並べ、それぞれ違う絵にのぼり着くことを見せる
- **絵の眼目**：ノイズを当てて引く一段を重ねると、でたらめな砂あらしから新しい絵が立ち上がる。出発の砂あらしの数だけ、絵がある

### ボディ4：言葉で、行き先を決める（画面5）

- **登場する図**：主役のキャンバス、誘導なしの分かれ道（複数の到達先候補）、プロンプト文字列「宇宙服を着たねこ」、方位磁針（針が行き先を指す）、のぼり階段、強度つまみ
- **レイアウト**：主役のキャンバスを中央やや左。方位磁針はキャンバスの右に、のぼる各段で参照される位置。プロンプト文字列は画面上部。強度つまみは方位磁針の下に横スライダーとして。誘導なしの分かれ道は、キャンバスの右がいくつかの到達先候補へ枝分かれする形で。のぼり階段はキャンバスの下。
- **動き**：砂あらしのキャンバスがのぼろうとするが、行き先が定まらず複数の到達先候補（ねこ／山／花）へふらふら枝分かれする（何が出るかは砂あらしまかせ）→ プロンプト文字列「宇宙服を着たねこ」が現れ、方位磁針へと姿を変える。針が「宇宙服のねこ」の方角を指す → のぼる一段ごとに方位磁針が参照され、枝分かれが一本の道へ束ねられていく。キャンバスが宇宙服のねこへ寄っていく → 強度つまみが現れる：弱に寄せると針の補正が弱まりキャンバスが注文から外れ気味に、強に寄せると針にきっちり従うが絵が硬くなる。つまみをほどよい位置へ置く → のぼりきって、キャンバスに宇宙服を着たねこのドット絵が結ぶ
- **絵の眼目**：プロンプトは絵そのものを持たず、方位磁針として、ノイズを引く一段ごとの向きを決める。その効き目も調節できる

### 結論（画面6）

- **登場する図**：主役のキャンバス、工程アイコンの帯（壊す／練習／のぼる／方位磁針）、砂あらし、虫めがね、完成した宇宙服のねこ、のぼり道の光の筋
- **レイアウト**：上に工程アイコンの帯。中央にキャンバス。最後はキャンバスに完成した絵を残して終わる。
- **動き**：工程アイコンが順に灯る：壊す → 練習 → のぼる → 方位磁針。アイコンが灯るのに合わせ、中央のキャンバスが全工程を早送りでなぞる（鮮明な絵 → 砂あらし → のぼって絵 → 宇宙服のねこ）→ 序論の不思議に回帰：キャンバスが砂あらしに戻り、虫めがねが一点に寄り、やはり粒しかないことを示す → だが「のぼる道のり」が光の筋として描かれ、その先に宇宙服のねこが結ぶ。キャンバスが完成した絵を映して静かに終わる
- **絵の眼目**：壊して・技を仕込んで・のぼって・言葉で舵を取る。全工程は、砂あらしの中に隠れた絵を探すのではなく、ノイズを引く道のりの先に絵を作り出す一本の道だった

## 4. 採否判断

- [x] script.md の6画面すべてに対応するデザインがある
- [x] 固定ベース（白テーマ・文字サイズ下限）を守り、比喩・図の語彙・アクセント色を拡散モデル固有に設計した（明るい現像台）
- [x] 背骨（キャンバスのノイズ量）が一本通り、装置はそれを加工する一連の工程として連なる
- [x] 各形が「概念にラベルを貼る箱」ではなく「仕組みを動いて見せる専用装置」になっている
- [x] 各画面に「絵の眼目」（絵だけで伝わる中心概念）があり、画面内で図が状態変化する
- [x] 「何もない所から湧く／消える」がない：階段の段は主役キャンバスの写し取り、記録ノイズ片は階段の各段から、予測ノイズ片はノイズ予測器から、方位磁針はプロンプト文字列から姿を変える
