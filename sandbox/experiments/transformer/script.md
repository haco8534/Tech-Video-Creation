# Transformer ── AIはどうやって文章を読んでいるのか

# ▶ 序論

めたん：ねえずんだもん。スマホで文字を打っていると、次に来る言葉を勝手に予想してくれますわよね。
ずんだもん：あるのだ。「おはよう」と打つと「ございます」が候補に出てくる、あれなのだ。
めたん：そうそれですわ。地味ですけれど、よく考えると不思議ではありませんこと？機械はどうして「次に来る言葉」が分かるのかしら。
ずんだもん：いい疑問なのだ。しかもその「次の言葉を当てる」力は、予測変換だけの話じゃないのだ。
めたん：と、おっしゃいますと？
ずんだもん：外国語をなめらかに訳す自動翻訳も、質問に文章で答えてくれる ChatGPT も──根っこは全部おなじなのだ。「ここまでの流れなら、次はこの言葉だ」を、ひたすら当てているのだ。
めたん：あんなに違って見えるものが、おなじひとつの仕組み……。
ずんだもん：そうなのだ。そしてその心臓にあたる仕組みには、ちゃんと名前があるのだ。Transformer というのだ。
めたん：トランスフォーマー。
ずんだもん：2017年に発表されて、それからのAIの世界を、まるごと塗り替えた仕組みなのだ。今のAIブームは、ここから始まったと言ってもいいのだ。
めたん：その Transformer が、今日のお話ですのね。
ずんだもん：そうなのだ。「機械はどうやって文章を読んでいるのか」──それを、絵で一歩ずつ追いかけていくのだ。
めたん：でも、文章を読むのって、そんなに大したことですの？わたくしたち、毎日当たり前にやっていますけれど。
ずんだもん：それなのだ。人間には当たり前すぎて気づかないけど、「言葉の意味を取る」のは、機械にとって、ずっと難問だったのだ。
めたん：あら。では、まずはその「難しさ」から、教えてくださいまし。

# ▶ ボディ

## ことばの意味は、まわりが決める

ずんだもん：たとえば、「かける」という言葉を、ひとつだけ取り出してみるのだ。
めたん：「かける」……ずいぶん、いろいろな意味がありそうですわね。
ずんだもん：「めがねをかける」。「いすに腰をかける」。「友だちに電話をかける」。「保険をかける」。
めたん：ぜんぶ「かける」ですのに、意味はてんでバラバラですわ。
ずんだもん：そうなのだ。じゃあ聞くのだ。「かける」という言葉だけを、ぽつんと見せられて、どの意味か分かるのだ？
めたん：……分かりませんわ。「かける」だけでは、なんとも。
ずんだもん：そうなのだ。「かける」単独では、意味が決まらないのだ。決めているのは「めがね」や「電話」──まわりにいる言葉なのだ。
めたん：まわりの言葉が、その言葉の意味を、決めている。
ずんだもん：人間はこれを一瞬でやっているのだ。「電話を」と来た時点で、もう「かける」の意味を、電話用にしぼっているのだ。
めたん：言われてみれば、無意識にやっていますわね。
ずんだもん：英語でも同じなのだ。「bank」という単語は、「銀行」にも「川の岸」にもなるのだ。
めたん：それも、まわりを見ないと決まりませんわね。お金の話か、川の話か。
ずんだもん：どんな言語でも、言葉は単独では、意味が揺れているのだ。まわりとセットで、はじめて意味が定まるのだ。
めたん：それに、ひとつひとつの言葉の意味が分かっても、まだ足りない気がしますわ。「ねこが いぬを 追う」と「いぬが ねこを 追う」では──
ずんだもん：おっと、それも大事なところなのだ。使っている言葉は全部おなじなのに、だれがだれを追うのかは、正反対なのだ。
めたん：言葉を、ただ集めるだけではだめ。どの言葉と、どの言葉が、どうつながっているか、まで読まないと。
ずんだもん：そういうことなのだ。文章を読むというのは、言葉そのものと、言葉どうしのつながり、その両方を取ることなのだ。
めたん：でも、昔の機械はそれを、どうしていましたの？
ずんだもん：ここが苦しいところだったのだ。昔の機械は、ひとつの言葉に、ひとつの決まった意味を、あてがっていたのだ。
めたん：「かける」には「かける」用の、たった一個の意味を。
ずんだもん：そう。分厚い辞書を一冊持っていて、言葉が来たら引く。それだけなのだ。
めたん：それだと、「めがね」の「かける」と「電話」の「かける」が、おなじになってしまいますわ。
ずんだもん：区別がつかないのだ。だから機械の文章読解は、長いあいだ、そこで足踏みしていたのだ。
めたん：言葉を、まわりから切り離して、見ていたから。
ずんだもん：そうなのだ。だから次に必要なのは、こういう仕組みなのだ──ひとつひとつの言葉が、自分でまわりを見わたして、自分の意味を決められること。
めたん：その仕組みこそが、Transformer の入り口、というわけですわね。

## 単語どうしが、見つめ合う

ずんだもん：ここからが、Transformer の心臓部なのだ。名前は Self-Attention。日本語にすると「自己注意」なのだ。
めたん：自己注意。言葉が、自分たちで、注意を向け合う……？
ずんだもん：イメージはこうなのだ。文章をならべると、ひとつひとつの言葉が、ほかの全部の言葉に向かって、こう問いかけるのだ。「きみは、ぼくの意味に関係ある？」
めたん：文章じゅうの言葉が、おたがいに、問いかけ合う。
ずんだもん：そして相手ごとに「これくらい関係ある」という強さを、見積もるのだ。これが「注目」なのだ。
めたん：さっきの「電話をかける」なら──
ずんだもん：「かける」は「電話」に強く注目するのだ。逆に「を」みたいな言葉には、ほとんど注目しないのだ。
めたん：注目に、強い弱いの差が、つくのですわね。
ずんだもん：そこが大事なのだ。全部に同じだけ注目するんじゃない。持っている注目の量は決まっていて、それを、関係の深い相手に、ぐっと多く配るのだ。
めたん：注目を、配分する。でも、その「どれくらい関係あるか」は、どうやって決めますの？相手の心が読めるわけでも、ありませんでしょう。
ずんだもん：いい質問なのだ。ここで、言葉はそれぞれ、三つの顔を持つのだ。Query、Key、Value──質問、鍵、中身、の三役なのだ。
めたん：ひとつの言葉が、三役を、同時に持つ。
ずんだもん：まず Query。これは「ぼくは、こういう情報をさがしている」という、問いかけの顔なのだ。
めたん：さがしものの内容を書いた、ふだ。
ずんだもん：次に Key。これは「わたしは、こういう情報を持っているよ」という、名札の顔なのだ。
めたん：問いかけと、名札。
ずんだもん：ある言葉の Query と、相手の言葉の Key。この二つを照らし合わせて、ぴたっと噛み合うほど──その相手への注目が、強くなるのだ。
めたん：「かける」の Query が「動作の相手をさがしている」で、「電話」の Key が「わたしは相手になれる名詞だよ」なら──
ずんだもん：見事に噛み合うのだ。だから「かける」は「電話」に強く注目する。逆に「を」の Key とは噛み合わないから、注目は弱いままなのだ。
めたん：なるほど。問いと名札の相性で、注目の配分が決まる。では、三つめの Value は？
ずんだもん：Value は「中身」なのだ。注目の配分が決まったら、最後に相手の Value──持っている中身そのものを、注目の強さのぶんだけ、受け取るのだ。
めたん：強く注目した相手からは、中身をたっぷり。弱い相手からは、ちょっぴり。
ずんだもん：そのとおりなのだ。そうやって、ひとつの言葉が、文章じゅうの言葉から、中身を少しずつ集めて、混ぜ合わせるのだ。
めたん：注目の配分どおりに、ブレンドしていく。
ずんだもん：そして集め終わったとき──その言葉は、もう最初の「ただの単語」では、ないのだ。
めたん：まわりの文脈を、自分の中に、取り込んでいる。
ずんだもん：「かける」は、もう「電話の文脈を着こんだ、かける」になっているのだ。意味が、ちゃんとしぼられているのだ。
めたん：あの、単独では意味が揺れていた言葉が、まわりと混ざって、定まりましたわ。
ずんだもん：これが Self-Attention の、一回ぶんの仕事なのだ。そして文章じゅうの言葉が、いっせいに、これをやるのだ。
めたん：全員が、同時に。それぞれ自分のまわりを見わたして、文脈を着こむ。
めたん：……でも、ひとつ疑問が。その Query や Key の中身は、だれが決めますの？「電話は名詞ですよ」と、人間が教えてあげるのかしら。
ずんだもん：そこなのだ。じつは、人間は教えないのだ。機械が、自分で見つけるのだ。
めたん：自分で？
ずんだもん：膨大な文章を読みながら、「どんな Query と Key を作れば、文章の意味がうまく取れるか」を、すこしずつ調整していくのだ。最初は、まるででたらめなのだ。
めたん：読めば読むほど、三つの顔が、上手になっていく。
ずんだもん：そうなのだ。だれかが正解の辞書を渡すんじゃない。大量の文章そのものが、先生なのだ。
めたん：Transformer は、仕組みのかたちだけ用意されていて、中身は、自分で学ぶのですわね。
ずんだもん：そういうことなのだ。Transformer が「文章を読む」というのは、つきつめると、この Self-Attention なのだ。

## 視点を増やし、積み重ねる

めたん：でも、ずんだもん。「関係がある」と言っても、いろいろな関係が、ありませんこと？
ずんだもん：よく気づいたのだ。たとえば「文法のつながり」で見たいときもあれば、「話の意味」で見たいときも、あるのだ。
めたん：「だれが、何をした」という骨組みと、「どんな雰囲気の話か」という中身は、別ものですものね。
ずんだもん：見たい関係が、一通りじゃないのだ。だから、注目の仕組みをひとつだけ持っていても、足りないのだ。
めたん：では、どうしますの？
ずんだもん：かんたんなのだ。おなじ Self-Attention を、何セットも用意して、同時に走らせるのだ。これを Multi-Head Attention──「複数の頭」と呼ぶのだ。
めたん：頭がいくつもあって、それぞれ違う見方で、おなじ文章を、ながめている。
ずんだもん：ひとつの頭は、主語と述語のつながりを追いかける。別の頭は、言いかえや、指ししめす関係を追いかける。そうやって、手分けするのだ。
めたん：おおぜいの読み手が、それぞれ違う色のペンで、おなじ文章に、線を引いているような。
ずんだもん：いい例えなのだ。一本のペンでは、一種類の関係しか引けない。でも何色もあれば、文章にひそむ関係を、いろんな角度から、すくい取れるのだ。
めたん：そして最後に、その線を全部かさね合わせて、ひとつの理解に、まとめる。では、それで文章を読み終わり、ですの？
ずんだもん：まだなのだ。ここでもうひとつ、Transformer のうまいところが、出てくるのだ。
めたん：と、おっしゃいますと？
ずんだもん：複数の頭で「文脈を着こんだ言葉たち」が、できたのだ。その言葉たちを──もう一度、おなじ仕組みに、まるごと通すのだ。
めたん：一度文脈を取り込んだ言葉を、さらにもう一度、注目し合わせる。
ずんだもん：そうなのだ。Self-Attention の層を、何段も積み重ねるのだ。下の階で出てきた結果が、そのまま、上の階の入り口になるのだ。
めたん：階を上がるごとに、何が変わりますの？
ずんだもん：浅い階では、となりの言葉とのつながりみたいな、こまかい関係をつかむ。階を上がるほど、文ぜんたいで何を言いたいのか、という大きな意味を、つかむのだ。
めたん：細部から、全体へ。階段をのぼるように、理解が深くなっていく。
ずんだもん：そのとおりなのだ。一階ぶんでは小さな変化でも、何十階も積めば、ただの単語の列が、深い意味のかたまりに、育つのだ。
めたん：たくさんの頭で、何層も。ずいぶん、大がかりになってきましたわね。
ずんだもん：そうなのだ。でも、これでもまだ、ひとつ抜けているものが、あるのだ。
めたん：……ひとつ、わたくしも、気になっていたことがありますの。
ずんだもん：なんなのだ？
めたん：Self-Attention は、全部の言葉を、いっぺんに見わたすのですわよね。それだと、言葉の「順番」は、どこへ行ってしまいますの？
ずんだもん：おおっ、それなのだ。まさにそれが、抜けているものなのだ。注目し合うだけだと、順番の情報が、すっぽり抜け落ちるのだ。
めたん：「ねこが さかなを 食べた」と「さかなが ねこを 食べた」が、同じに見えてしまいますわ。
ずんだもん：それは、大事件なのだ。意味が正反対なのに、区別できないのだ。
めたん：では、どうやって、順番を教えますの？
ずんだもん：言葉ひとつひとつに、「きみは何番目」という順番の目印を、こっそり付け足してやるのだ。これを、位置エンコーディングと呼ぶのだ。
めたん：注目で意味を集めて、目印で順番を守る。
ずんだもん：その両方がそろって、はじめて、ばらばらの単語が「ちゃんとした文章」として、読めるのだ。

## なぜ Transformer は世界を変えたのか

めたん：ねえずんだもん。Transformer の前にも、機械は文章を読んでいましたわよね。何が、そんなに変わりましたの？
ずんだもん：そこを話して、しめくくるのだ。Transformer の前、主役だったのは、RNN という仕組みなのだ。
めたん：アールエヌエヌ。
ずんだもん：RNN は、文章を「前から一語ずつ」、順番に読んでいくのだ。
めたん：わたくしたちが本を読むのと、おなじですわね。一語ずつ、前から。
ずんだもん：素直なやり方なのだ。でも、一語ずつだと、こまったことが、二つ起きるのだ。
めたん：二つ。まず、ひとつめは？
ずんだもん：とにかく、遅いのだ。前の言葉を読み終わらないと、次の言葉に、進めないのだ。
めたん：ひとり終わるまで、次の人が待っている。一列にならんだ、行列のような。
ずんだもん：そう。文章が長くなるほど、行列も長くなって、どんどん時間がかかるのだ。
めたん：では、Transformer は？
ずんだもん：Transformer の Self-Attention は、全部の言葉を「いっぺんに」見比べるのだ。順番待ちが、そもそも、無いのだ。
めたん：行列にならばずに、全員が、同時に動ける。
ずんだもん：何百語あっても、まとめて、並列に処理できる。これが、ひとつめの大きな違いなのだ。
めたん：では、ふたつめの、こまったことは？
ずんだもん：RNN は、情報をバケツリレーで、後ろへ送っていくのだ。一語、また一語と、手わたしで。
めたん：バケツリレー……長くなると、最初の水が、最後まで届きませんわね。
ずんだもん：そうなのだ。文章の最初のほうに出てきた大事な言葉が、後ろへ届くころには、うすれて、消えかけてしまうのだ。
めたん：遠くはなれた言葉どうしが、つながれない。
ずんだもん：でも Transformer は、注目で、いきなり直接つながるのだ。文の一語目と百語目でも、ひとっとびで、結べるのだ。
めたん：どれだけ離れていても、距離が、ゼロ。
ずんだもん：遠さが、関係なくなるのだ。これが、ふたつめの大きな違いなのだ。
めたん：速くて、遠くまで届く。……それが、あの巨大なAIに、つながるのですの？
ずんだもん：つながるのだ。ここで、いちばん最初の話を、思い出してほしいのだ。次の言葉を当てる、あれなのだ。
めたん：予測変換の。
ずんだもん：Transformer は、ああやって学ぶのだ。文章の途中の言葉をかくして、「次はなんだ」と当てさせる。外れたら、すこし直す。それを、とほうもない回数、くりかえすのだ。
めたん：次の言葉あてを、ひたすら、練習する。
ずんだもん：その練習に、本でいえば何千万冊ぶんもの文章を、使うのだ。そして並列に処理できるからこそ、その膨大な練習を、現実的な時間で、終わらせられるのだ。
めたん：RNN の行列のままでは、とても終わらなかった量を。
ずんだもん：そうなのだ。ChatGPT のような大きなAIは、この「並列に処理できる」性質の、上に建っているのだ。Transformer が、AIを大きくする扉を、開けたのだ。

# ▶ 結論

ずんだもん：ここまでを、絵といっしょに、振り返るのだ。
ずんだもん：まず──ことばの意味は、まわりが決める。
めたん：だから言葉どうしが注目し合って、まわりの文脈を、自分に着こむ。
ずんだもん：その注目を、いくつもの視点でおこなって、層として、何段も積み重ねる。順番の目印も、忘れずに。
めたん：そして全部いっぺんに見わたせるから──速くて、遠くまで届いて、そして、大きくできる。
ずんだもん：その全部をまとめた仕組みが、Transformer なのだ。
めたん：はじめのお話に、戻ってきますわね。スマホが、次の言葉を予想してくれるとき──
ずんだもん：その裏側では、打ったばかりの言葉たちが、いっせいに見つめ合って、文脈を、確かめ合っているのだ。
めたん：あの小さな予測変換の中で、こんなことが、起きていたなんて。
ずんだもん：翻訳のときも、ChatGPT が答えるときも、動いているのは、おなじ、この仕組みなのだ。
めたん：次に文字を打つときは、画面の中が、少し違って、見えそうですわ。
ずんだもん：機械が言葉を「見わたす」しくみ──Transformer のお話、だったのだ。
