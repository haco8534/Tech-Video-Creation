# YouTube概要欄：URLを打ち込んだ瞬間、何が起きるのか

<!-- 以下をそのままYouTubeの概要欄にコピー＆ペーストして使う -->

ブラウザにURLを入力してEnterキーを押す──毎日やっているこの操作の裏で、実は0.数秒の間に壮大なドラマが繰り広げられています。
DNS名前解決、TCPハンドシェイク、TLS暗号化、HTTPリクエスト、そしてブラウザのレンダリング。
5つのステップが精密に連鎖する「インターネットの裏側」を、ずんだもんとめたんがわかりやすく解説します。
