# YouTube概要欄：暗号化とハッシュ化——似ているようで全く違う

<!-- 以下をそのままYouTubeの概要欄にコピー＆ペーストして使う -->

「パスワードは暗号化して保存されている」と思っていませんか？
実はそれ、大きな誤解です。暗号化とハッシュ化は、名前は似ていても目的も仕組みも正反対。
この動画では、暗号化は「鍵のかかった箱」、ハッシュ化は「指紋」というアナロジーで、2つの決定的な違いをわかりやすく解説します。
Adobe 1.53億アカウント流出事件やLinkedIn漏洩事件の教訓から、なぜパスワードはハッシュ化すべきなのかを明らかにします。
